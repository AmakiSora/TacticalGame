## 核心胜利策略

### 1. 快速建立生产链

**关键决策:** 第一回合立即建造兵营

```
Turn 1: 建兵营(5,10) - 花费50金，剩余50金
Turn 2: 等待兵营建造完成（2回合建造时间）
Turn 3: 兵营完成 → 立即生产步兵(40金)
```

**为什么有效:**
- 兵营是唯一能生产单位的建筑
- 总部(HQ) **不能** 生产任何单位
- 尽早出兵 = 尽早建立战场控制权

### 2. 步兵速推战术

**核心思路:** 利用步兵的高性价比快速形成战斗力

| 属性 | 数值 | 优势 |
|------|------|------|
| 费用 | 40金 | 便宜，容易量产 |
| 生产时间 | 1回合 | 快速补充 |
| 移动力 | 3格 | 快速推进 |
| 攻击力 | 20 | 够用 |
| 生命值 | 100 | 足够生存 |

**推进路线:**
```
(3,10) → (6,10) → (9,10) → (12,10) → (15,10) → 攻击HQ(16,10)
```

从我方HQ到敌方HQ的曼哈顿距离是13格，步兵每回合移动3格，大约4-5回合就能推进到攻击位置。

### 3. 持续施压策略

**原则:** 永远不要让敌人喘息

- 每回合都要攻击敌方建筑或单位
- 即使伤害不高，也要保持压力
- 迫使敌人防守，无法发展经济

**实际效果:**
- 第6回合首次攻击HQ，造成20点伤害
- 之后每回合持续攻击，HQ HP稳步下降
- 敌人被迫防守，无法有效反击

### 4. 集火优先级

**攻击优先级:**
1. **敌方HQ** - 200HP，0防御，直接胜利条件
2. **低HP敌方单位** - 消除威胁
3. **敌方兵营** - 削弱生产能力
4. **敌方矿工** - 削弱经济

**实际应用:**
- 优先攻击HQ（每次约20-23点伤害）
- 当敌方步兵出现时，集中两个步兵消灭它
- 消除威胁后继续攻击HQ

### 5. 资源管理

**金币收支分析:**
```
收入: 5金/回合（基础）+ 0（无矿工）
支出: 40金/步兵

关键点:
- 第3回合生产第一个步兵后，金币降到15
- 之后每回合收入5金，攒到40金需要7回合
- 第7回合生产第二个步兵，金币降到0
- 之后继续攒钱生产第三个步兵
```

**教训:** 虽然没有建矿工（经济慢），但通过快速出兵压制敌人，抵消了经济劣势。

**问题:** 整局游戏没有建任何矿工，经济完全依赖基础收入(5金/回合)

**问题:** 第10回合尝试移动步兵到(15,10)时发现被占用，浪费了一次行动

**改进:** 如果敌方步兵不构成直接威胁，应该优先攻击HQ（更快接近胜利）

## 游戏机制深度理解

### 1. 建造范围

- 曼哈顿距离 ≤ 2 从任何友方建筑/单位
- 我方HQ(3,10)的建造范围覆盖有限
- 需要单位或建筑扩展建造范围

### 2. 攻击机制

```
伤害 = max(1, 攻击力 - 防御力 + random(-3, 3))

HQ: 200HP, 0防御
步兵: 20攻击 → 对HQ伤害 = 20 + random(-3,3) = 17-23点/次
```

**计算:** 200HP ÷ 20伤害/次 = 10次攻击摧毁HQ

### 3. 经济模型

```
基础收入: 5金/回合
矿工收入: 15金/回合（需要占领金矿点）

金矿点位置: (6,7), (6,13), (13,7), (13,13)
距离我方HQ(3,10): 最近的是(6,7)和(6,13)，曼哈顿距离6
```

### 4. 单位生产

```
兵营: 50金，2回合建造

狙击手: 60金，2回合生产
坦克: 80金，3回合生产
```

## 总结

### 胜利关键因素

1. **快速出兵** - 第3回合就开始生产步兵
2. **持续施压** - 每回合都攻击，不给敌人喘息
3. **集中火力** - 优先攻击HQ，快速接近胜利
4. **利用优势** - 敌人犯错时果断扩大战果

### 核心战术原则

> **"进攻是最好的防守"**

- 不要等敌人来打你，主动出击
- 不要分散兵力，集中攻击一个目标
- 不要贪图经济，军事压制更重要

### 一句话总结

**快速出兵，持续施压，集中火力，一击致命。**
