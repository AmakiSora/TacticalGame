## 核心失败原因

### 1. 没有尽早占领采矿点

**错误:** 整局游戏没有建造任何矿工，经济完全依赖基础收入(5金/回合)

**后果:** 
- 收入严重落后：我每回合只有5金，而对手有20+金（矿工收入）
- 无法快速生产单位：攒到40金需要8回合，对手只需要2回合
- 兵力差距越来越大：对手有5-6个步兵，我只有1-2个

**正确做法:**
```
Turn 1: 建兵营(5,10) → 剩余50金
Turn 2: 移动步兵到(6,7)附近 → 建矿工(30金) → 剩余25金
Turn 3: 兵营完成 → 生产步兵(40金) → 矿工完成，开始产出+15金/回合
Turn 4+: 收入: 5基础 + 15矿工 = 20金/回合
```

### 2. 被动防守，没有主动进攻

**错误:** 一直在攻击敌方步兵，没有向敌方总部推进

**后果:**
- 浪费了大量回合在小规模战斗上
- 没有对敌方总部造成任何伤害
- 敌方可以安心发展经济和生产单位

**正确做法:**
- 优先攻击敌方总部（200HP，0防御）
- 不要和敌方步兵纠缠，直接绕过去攻击总部
- 集中火力，快速结束战斗

### 3. 没有保护总部

**错误:** 让敌方步兵直接攻击我的总部

**后果:**
- 总部 HP 从 200 降到 36，最终被摧毁
- 失去了生产建筑和经济基础

**正确做法:**
- 在总部附近保留防御单位
- 及时回防，阻止敌方步兵靠近总部
- 建造第二个兵营，增加生产速度
