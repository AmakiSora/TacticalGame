## 核心胜利策略

### 1. 经济优先 - 尽早建矿工

**关键决策:** 第5回合建造矿工

```
Turn 1: 建兵营(5,10) - 花费50金，剩余50金
Turn 2: 等待兵营建造完成（2回合建造时间）
Turn 3: 兵营完成 → 立即生产步兵(40金)
Turn 4: 移动步兵到(6,8)，接近采矿点(6,7)
Turn 5: 建矿工(6,7) - 花费30金，剩余0金
```

**为什么有效:**
- 矿工收入是基础收入的3倍（15金 vs 5金）
- 越早建矿工，经济优势越大
- 经济是军事的基础，没有足够的资源就无法维持防线

**实际效果:**
- 第5回合建矿工后，收入从5金/回合提升到20金/回合
- 之后每回合都能生产步兵（40金）
- 兵力持续增加，最终形成数量优势

### 2. 步兵速推战术

**核心思路:** 利用步兵的高性价比快速形成战斗力

| 属性 | 数值 | 优势 |
|------|------|------|
| 费用 | 40金 | 便宜，容易量产 |
| 生产时间 | 1回合 | 快速补充 |
| 移动力 | 3格 | 快速推进 |
| 攻击力 | 20 | 够用 |
| 生命值 | 100 | 足够生存 |

**推进路线:**
```
(3,10) → (6,8) → (9,8) → (12,8) → (15,10) → 攻击HQ(16,10)
```

从我方HQ到敌方HQ的曼哈顿距离是13格，步兵每回合移动3格，大约4-5回合就能推进到攻击位置。

### 3. 持续施压策略

**原则:** 永远不要让敌人喘息

- 每回合都要攻击敌方建筑或单位
- 即使伤害不高，也要保持压力
- 迫使敌人防守，无法发展经济

**实际效果:**
- 第8回合首次攻击敌方步兵，造成11点伤害
- 之后每回合持续攻击，敌方步兵HP稳步下降
- 第18回合开始攻击敌方总部，造成45点伤害
- 第22回合摧毁敌方总部，获得胜利

### 4. 集火优先级

**攻击优先级:**
1. **敌方HQ** - 200HP，0防御，直接胜利条件
2. **低HP敌方单位** - 消除威胁
3. **敌方兵营** - 削弱生产能力
4. **敌方矿工** - 削弱经济

**实际应用:**
- 优先攻击HQ（每次约20-23点伤害）
- 当敌方步兵出现时，集中攻击它
- 消除威胁后继续攻击HQ

### 5. 让所有单位行动

**原则:** 每个单位每回合都可以行动一次，不要浪费

- 移动步兵向敌方推进
- 攻击敌方建筑或单位
- 调整位置，为下回合做准备

**实际效果:**
- 每回合都让所有步兵行动
- 兵力持续增加，最终形成数量优势
- 不浪费任何一次行动机会

## 游戏机制深度理解

### 1. 经济的重要性

```
基础收入: 5金/回合
矿工收入: 15金/回合（需要占领金矿点）

关键点:
- 矿工收入是基础收入的3倍
- 越早建矿工，经济优势越大
- 经济是军事的基础，没有足够的资源就无法维持防线
```

### 2. 采矿点位置

```
金矿点位置: (6,7), (6,13), (13,7), (13,13)

距离我方HQ(3,10):
- (6,7): 曼哈顿距离 6
- (6,13): 曼哈顿距离 6
- (13,7): 曼哈顿距离 13
- (13,13): 曼哈顿距离 13

建议: 优先占领 (6,7) 或 (6,13)，因为它们离我方HQ更近
```

### 3. 建造范围

- 曼哈顿距离 ≤ 2 从任何友方建筑/单位
- 需要先派单位靠近采矿点，然后才能建矿工
- 矿工建造时间: 1回合
- 矿工成本: 30金

### 4. 攻击机制

```
伤害 = max(1, 攻击力 - 防御力 + random(-3, 3))

HQ: 200HP, 0防御
步兵: 20攻击 → 对HQ伤害 = 20 + random(-3,3) = 17-23点/次

计算: 200HP ÷ 20伤害/次 = 10次攻击摧毁HQ
```

### 5. 单位生产

```
兵营: 50金，2回合建造
步兵: 40金，1回合生产
狙击手: 60金，2回合生产
坦克: 80金，3回合生产
医疗兵: 50金，1回合生产
```

1. **经济优先** - 尽早建矿工，收入从5金/回合提升到20金/回合

- 经济领先 = 兵力领先 = 胜利
