## 关键决策与经验总结

### 1. 🏗️ 前期：经济优先，用墙扩展建造范围 (Turn 1-6)

**决策：**
- Turn 1: 建兵营 (4,10) — 花50金
- Turn 2: 建墙 (5,7) — 向矿点扩展建造范围
- Turn 3: 建矿工 (6,7) — 第一台矿机上线！
- Turn 4: 建墙 (5,13) — 向另一个矿点扩展
- Turn 5: 建矿工 (6,13) — 第二台矿机上线！
- Turn 6: 建墙 (7,7) — 继续向东推进

**经验：** 矿工是经济的命脉。Player B 第一回合花100金建了两个兵营，导致经济落后。我用墙扩展建造范围后建了2台矿机，收入达到35金/回合（基础5 + 矿工15×2），而 Player B 只有一个矿工收入20金/回合。**经济优势是胜利的基石。**

### 2. 🪖 爆兵策略：纯步兵海 (Turn 7-15)

**决策：**
- 全程只出步兵（40金，1回合生产），不造狙击手/坦克/医疗兵
- 每回合只要有40金就产一个步兵，保持持续暴兵
- 最终共生产 **10个步兵**（Player B 生产了约 5-6个）

**经验：** 在有限的经济下，步兵海战术最有效。低级单位数量优势可以弥补个体战斗力不足。利用步兵3格的移动力和1格攻击范围，可以灵活包围和集火。

### 3. ⚔️ 战斗技巧：集火与包围 (Turn 8-15)

**关键战斗记录：**

| 回合 | 事件 |
|------|------|
| T8 | 第一次交火，双方步兵在(8,9)附近交战 |
| T9 | Player B 步兵攻击我方(8,9)，我方反击 |
| T10 | 激烈混战，双方互相攻击 |
| **T11** | **Player B 第一个步兵阵亡 (8,9)** |

**经验：**
- 集火战术极为有效。在 T15 我三个步兵同时攻击一个敌方步兵（分别从 (9,8)、(8,9)、(7,8) 三个方向），一回合内击杀。
- 攻击后可移动的机制非常灵活：先攻击再移动到更有利的位置，或先移动到攻击位置再攻击。
- 数值优势：步兵攻击力40，敌方防御8，每次造成约32-35伤害。3个步兵集火 = 约96-105伤害，足够击杀100HP的步兵。

### 4. 🏃 快速推进与斩首 (Turn 16-19)

| 回合 | 事件 |
|------|------|
| T16 | 推进至(12,8)、(13,10)，逼近敌方基地 |
| **T17** | **首次攻击HQ！(15,10) → HQ 40伤害 (160/200)** |
| **T18** | **再次攻击HQ！43+41=84伤害 (76/200)，步兵(16,11)进场** |
| **T19** | **最后一击！38+40=78伤害，HQ 被摧毁！** |

**经验：**
- 当获得兵力优势后，不要浪费时间去清理所有敌方单位。直接冲向HQ执行"斩首"战术。
- 包围HQ的攻击位置：(15,10)、(16,11) 是关键攻击位，可以从这两个位置同时攻击HQ。
- 即使是残血步兵（2HP）也能打出满额伤害（40），不要低估残血单位的价值。

### 5. 🧱 墙的战术价值

建了5堵墙扩展建造范围：
- (5,7)、(5,13) — 连接矿点
- (7,7)、(9,7)、(11,7) — 向东推进的跳板

**经验：** 墙不仅可以防御，更重要的是可以像"跳板"一样扩展建造范围，让矿工和兵营可以建在更远的位置。

## 反思

### 做得好 👍
1. **经济先行** — 2台矿机带来的经济优势是获胜的根本
2. **持续暴兵** — 每回合产兵，从不间断
3. **集火战术** — 多打一，快速消灭敌方有生力量
4. **目标明确** — 取得优势后直奔HQ
5. **墙的扩展** — 合理利用墙扩展建造范围

### 可以改进 🔧
1. **前期出兵偏晚** — T7才出第一个步兵，可以更早开始暴兵
2. **没有造第二个兵营** — 只有一个兵营，每回合只能产一个步兵
3. **没有使用其他兵种** — 狙击手(60金、2回合)和坦克(80金、3回合)可能在后期更有用
4. **墙体防御不足** — 只建了单向墙，没有形成完整防线
