**问题:** 多个单位分散在地图各处，进攻效率不高
**改进:** 尽早将作战单位集中到进攻路线上
**收益:** 形成更强的攻击集团，提高伤害输出

**问题:** 部分单位在移动过程中错失攻击机会

**收益:** 最大化每个行动点的价值

## 游戏机制深度理解

### 1. 行动点限制
```
每回合: 5个行动点
每个单位激活: 消耗1个行动点
已激活单位: 可免费执行其他操作（如移动后攻击）
```

### 2. Hex V2移动机制
```
六边形距离: max(|dq|, |dr|, |ds|) 其中 s = -q-r
地形限制: 水域、障碍物、单位阻挡
路径规划: 需要绕过障碍物和水域
```

### 3. 部署机制
```
部署位置: 只能在己方控制点相邻空地
部署消耗: scout=40, infantry=45, heavy=90
部署时机: 应在有足够行动点时部署
```
