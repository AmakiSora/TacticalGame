## 核心胜利策略

### 1. 开局双scout抢三线

**关键决策:** 第1回合花光80补给部署2个scout，同时初始三单位分别指向西点、中央、北/南路线。

```
Turn 1: infantry -> cp_a(-4,0)，部署scout(-7,0)、scout(-7,-1)
Turn 2: scout -> cp_c(0,0)，另两只scout分别去(-5,3)、(-2,-4)
Turn 3-4: 占cp_n(0,-4)、cp_s(0,4)
```

**为什么有效:**
- scout移动5格，能比步兵更早进入中央和边路点。
- 第4回合我方已经控制cp_a/cp_c/cp_n/cp_s四点，收入达到70/回合。
- 对手虽拿到cp_b，但长期被限制在东侧，无法阻止我方多点展开。

### 2. 先控经济，再转HQ

**关键决策:** 第6回合用scout从南线绕到cp_b(4,0)，短暂形成五据点全控。

```
Turn 4: 控4点，收入70
Turn 6: scout(1,1)->(4,0)，夺cp_b
Turn 6结束: 五据点全控，下一次收入85
```

**为什么有效:**
- 五据点收入=10基础+75据点=85/回合，足以每回合部署ranger/heavy。
- 即使第9回合后对手夺回cp_b/cp_c，我方已经把经济转化为东侧ranger群。
- 第6-8回合的经济峰值直接购买了heavy和两只ranger，是终局火力来源。

### 3. Ranger海打HQ，忽略不必要清场

**关键决策:** 第8回合开始不再执着清中线残兵，而是让scout贴HQ、ranger在(5,0)(5,1)建立射击阵地。

```

Turn 10: HQ 191->119
```

**为什么有效:**

- 两只ranger加一只scout每回合可稳定输出70+，不需要完全清空战场。
- 对手反占据点发生在第10-12回合，但HQ已进入斩杀线，经济反扑来不及转化为防守。

### 4. Heavy只需要扛线，不一定负责输出

**关键决策:** 第6回合部署heavy(0,1)，后续虽多次被路径卡住，但仍吸收大量中线伤害。

```

```

**为什么有效:**
- heavy 160HP/16防，适合拖住中央敌军。
- 它没有贡献关键击杀，但拖住敌方步兵和行动点，让东侧ranger安全输出HQ。
- 对比纯scout/ranger，heavy提供了中线容错。

## 失误与改进

### 失误1: 多次误判六角距离和路径

**问题:** 第1-3回合曾把(-7,1)->(-4,2)误判为3步，实际六角距离为4；第7-11回合也多次让heavy或南线ranger走到不可达坐标。
**改进:** 每次移动前用 `max(abs(dq),abs(dr),abs(ds))` 先算理论距离，再考虑water/blocker/unit/HQ占位。
**收益:** 减少无效API尝试，实战中可少浪费思考时间和避免错过更好路径。

### 失误2: Support部署后治疗半径利用不足

**问题:** support在(-1,1)只能治疗距离1目标，第5回合想治疗(1,0)步兵失败，随后support很快被ranger击杀。
**改进:** support应紧贴主力两格后方，优先站在(0,1)或(0,0)附近；若目标距离2，应先移动support再治疗同一回合。
**收益:** 能保住40HP步兵和24HP scout，减少中线阵亡。

### 失误3: 中央守点薄弱，后期被反占

**问题:** 第10-12回合对手连续夺回cp_c/cp_b/cp_s/cp_a，说明我方过早把注意力转到HQ，没有留单位守点。
**改进:** 在第8回合部署ranger时，应同时让heavy留在cp_c附近，或额外部署infantry守cp_c/cp_s。
**收益:** 即使HQ战拖一回合，也能保持70-85收入，不给对手恢复到80补给。

### 失误4: 没有充分保护东侧低血ranger

**问题:** ranger(5,0)在第12回合只剩20HP，若被提前击杀，HQ会残血多拖一轮。
**改进:** 第二只ranger应更早移动到(5,1)，形成双ranger互相分担威胁；也可在cp_b附近部署support或heavy吸收守军攻击。
**收益:** 终局火力更稳定，不依赖残血单位完成斩杀。

### 1. 经济模型

```
基础收入: 10补给/回合
每个补给点: +15补给/回合

  总收入 635

```

结论：五点全控不是为了长期持有，而是为了把第6-8回合的经济窗口转化为ranger/heavy火力。

### 2. 攻击与终结

```
ranger: 42攻，射程3，打HQ约31-35伤害
scout: 18攻，射程1，打HQ约6-10伤害
HQ: 240HP，10防

本局HQ伤害:
  攻击12次，合计272伤害

```

结论：ranger是终局效率最高单位；scout负责贴脸补伤害和占点。

### 3. 行动点机制

```
每回合5行动点
部署必花1点
同一单位移动后攻击不额外花新行动点
```

结论：终局应优先激活能移动后攻击HQ的单位；新部署ranger本回合不能行动，只是下一回合火力投资。

## 总结

### 胜利关键因素

1. 第1-4回合快速占cp_a/cp_c/cp_n/cp_s，建立70收入。
2. 第6回合夺cp_b，短暂五点全控，把经济转化为heavy和ranger。
3. 第8回合果断转入HQ战，不被中线残兵拖住。
4. 两只东侧ranger加一只贴脸scout稳定输出，最终第12回合摧毁HQ。

### 核心战术原则

> “前四回合抢经济，第八回合转斩杀。”

- scout负责抢点和贴HQ。
- ranger负责终局输出。
- heavy负责拖住中线，不必强求击杀。
- support必须贴近主力，否则治疗射程1会变成摆设。

### 一句话总结

**本局胜利来自早期多点经济和中期ranger转HQ的果断切换；即使后期据点被反夺，只要HQ火力窗口已经建立，就能在对手经济反扑前结束比赛。**
