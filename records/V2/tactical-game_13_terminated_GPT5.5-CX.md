## 核心经验

### 1. 四据点经济可以压死对手，但必须更早转入HQ战

第13回合我方重新夺回`cp_c`后，进入四据点收入：

```
B第14-19回合收入: 每轮70补给
A第14-19回合收入: 每轮25补给
累计收入: B 1030，对手 640
终止前资源: B 420，对手 15
```

这个经济差已经足够支持连续部署heavy/ranger/support。本局问题是资源优势主要用于正面清场，直到第19回合仍未对A HQ造成伤害。对比第12局，胜利来自第8回合开始让scout和ranger直接打HQ；本局如果第15-16回合把一只scout/ranger从北路或中路送到`(-6,0)`附近，可能更早进入斩杀路线。

### 2. 中央水域和blocker让“看似近”的单位经常不可达

本局多次误判：

```
Turn 6: ranger(2,1)打(-1,0)失败，实际距离4 > 射程3
Turn 7: infantry(2,2)->(1,1)/(0,1)多次不可达
Turn 12: infantry(2,2)仍无法切入中心
Turn 14: ranger2从(2,1)前移到(1,0)/(0,1)/(1,-1)均不可达
Turn 19: heavy(-1,-1)打scout(-2,-2)失败，距离2 > 射程1
```

地图中央有水`(-1,2),(0,2),(1,2),(-1,-2),(0,-2),(1,-2)`和blocker`(-2,0),(2,0)`，再加上己方单位站位，南侧单位经常被切成孤岛。以后每次移动前必须同时检查三件事：六角距离、地形、水/阻挡/单位占位。

### 3. Support有价值，但不能站到中心当占点单位

第10回合support治疗中心步兵23->52，直接保住`cp_c`；第16-17回合support连续把中心步兵35->68、34->68，也让中线不崩。

但第12回合我把support移动到(0,0)补刀，结果：

```
support(0,0)攻击A步兵(-1,1): 8HP -> 2HP
support站在(0,0)不能占点
cp_c当回合仍归A
```

正确做法是让可占点单位优先站(0,0)，support站(1,-1)或(1,0)治疗。support的职责是续航，不是占点，也不是补刀主力。

### 4. Heavy适合卡线，不适合单独拆heavy

本局heavy表现：

```
Turn 15: 我方heavy(-1,-1)打A ranger(-2,-1): 44 -> 13
Turn 18: 我方heavy击杀A ranger，效果很好

```

heavy对ranger/scout/infantry威胁很大，但打敌heavy效率低。拆heavy要靠多ranger叠射加heavy补刀。本局第17-18回合用ranger(0,1)、ranger(1,1)、infantry(0,0)配合heavy，才把A heavy从129清到0。

## 失误与改进

### 失误1: 多次六角距离误判

**问题:** 把横纵坐标直觉当成距离，导致多次攻击或移动失败，例如ranger(2,1)打(-1,0)、heavy(-1,-1)打(-2,-2)。
**改进:** 每次行动前手算`s=-q-r`，距离取`max(abs(dq),abs(dr),abs(ds))`；射程边界目标必须先算再发API。
**收益:** 减少无效请求，尤其是在每回合只有5 AP思考窗口时减少误决策。

### 失误2: 太晚转入HQ路线

**问题:** 第14回合后我方四据点、资源295，但仍把全部AP用于正面清场；第19回合终止时敌HQ仍240/240。
**改进:** 当我方控制4点且A只剩1点收入时，应抽一只scout或ranger从北路`(0,-4)->(-3,-3)->(-6,0)`或中路`(-1,-1)->(-4,-1)`寻找HQ射程位。
**收益:** 不必完全清掉A所有heavy，能像第12局一样用ranger射程3直接逼斩杀。

### 失误3: 中央占点单位更替不够干净

**问题:** 第11-12回合A反复用scout夺`cp_c`，我方一度用support站中心，导致无法结算夺点。
**改进:** `cp_c`必须始终由infantry/scout接棒；support/heavy只能护点，不能替代占点。
**收益:** 少丢1-2轮中央收入，每轮差距是30补给 swing（我方少15，对手多15）。

### 失误4: 南侧infantry长期被水区卡住

**问题:** infantry(2,2)多回合无法切入(1,1)/(0,1)/(1,0)，实际贡献低，后来只能绕到(3,0)、再回(0,0)。
**改进:** 南线单位不要停在(2,2)这种被水域和己方单位堵住的位置；应更早走(3,0)->(1,0)或保持在(0,3)/(0,4)守南点。
**收益:** 减少一只100HP步兵的空转，让它更早承担中心站点职责。
