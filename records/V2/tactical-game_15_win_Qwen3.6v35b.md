
### 2. Scout快速突袭，持久HQ压制

- 敌方没有任何手段回应HQ威胁（无远程单位）

## 失误与改进

### 失误1: 早期未部署支援单位

**问题:** 整局未部署任何支援单位（support，60金）。在Turn 3-6期间，我方infantry在前线承受了敌方攻击（如infantry(3,-1)从100HP→37HP→14HP），如果有支援单位可以保持战线。

**改进:** 在Turn 3或Turn 5部署一个支援单位到(-7,1)附近，为前线提供治疗。

**收益:** 前线infantry可以保持100HP，避免Turn 13后只剩14HP的濒死状态。

### 失误2: 重坦部署时机错过

**问题:** 第14局（Qwen3.6v35b输了）同样因未充分利用兵种搭配而失败。本局尝试过但未部署重坦，因为HQ周围无法部署。如果能在其他位置建立部署点，重坦的36攻击可以更快清理敌方防线。

**改进:** 考虑部署到CP(-4,0)附近（如果地形允许），或接受当前兵种组合继续优化。

**收益:** 重坦可以在Turn 10前建立防线，减少我方单位损耗。

### 失误3: 侦察兵(8,-1)HP损耗过快

**问题:** scout(8,-1)从Turn 11开始持续攻击HQ，但HP从49逐步下降（无治疗）。如果该scout在Turn 19被击杀，将失去一个HQ攻击来源。

**改进:** 部署支援单位或让scout有节奏地轮换（如Turn 18不攻击HQ，先治疗再攻击）。

**收益:** 保持2个HQ攻击单位到最后，额外约16-24点HQ伤害。

## 游戏机制深度理解

### 1. 经济模型
```
基础收入: 10金/回合
CP收入: 15金/CP/回合（2CP=30金）
总收入: 40金/回合（双CP）
部署花费: infantry 45金，scout 40金
20回合总收入: 80初始 + 20×40 = 880金
20回合总支出: 170金（6单位部署）
最终剩余: 615金（含其他调整）
```

### 2. 六角网格距离
```
dist((q1,r1), (q2,r2)) = max(|dq|, |dr|, |ds|)
其中 s = -q-r, ds = |s1-s2|
示例: scout(-7,-1)→(8,0) HQ
  s1 = -(-7)-(-1) = 8, s2 = -(8)-0 = -8, ds = 16
  dist = max(15, 1, 16) = 16 (scout moveRange=5，需多回合)
```

### 3. 攻击机制
```
伤害 = max(1, 攻击力 - 防御力 + random(-3, 3))
HQ防御: 8
scout→HQ: max(1, 18-8+random(-3,3)) = 8~13/次
infantry→HQ: max(1, 28-8+random(-3,3)) = 18~28/次
20回合scout攻击≈11次×10.5 = 115（实际139含其他伤害来源）
```

### 4. 行动点限制
```
每回合5个行动点
部署=1AP，移动=1AP，攻击=1AP
已激活单位可免费移动（acted=true后）
最优策略：先攻击（消耗AP），再免费移动到新位置
```

## 总结

### 胜利关键因素
1. **早期双CP控制** — 第1-2回合抢占(-4,0)和(0,-4)，建立25金/回合额外收入
2. **Scout快速突袭HQ** — 利用5的移动范围，Turn 11即开始持续攻击敌方HQ
3. **避免无意义部署** — HQ周围无法部署时果断积累补给，20回合剩余615金
4. **持久消耗战** — 不急于决战，通过每回合8-13点HQ伤害逐步瓦解敌方

### 核心战术原则

> **"侦察先行，持久压制，HQ伤害是胜负核心"**

- 用scout的5格移动范围快速建立HQ威胁
- 每回合对HQ施加压力比击杀敌方单位更重要
- 经济优势（双CP）比兵力数量更关键
- 20回合游戏的核心是"累积伤害"而非"短期歼灭"

### 一句话总结

**"用侦察兵的机动性建立持续HQ压制，用双CP的经济优势支撑持久战，139点HQ伤害+满血我方HQ=压倒性胜利。"**
