
### 2. 攻击机制
```
伤害 = max(1, 攻击力 - 防御力 + random(-3, 3))
游骑兵46攻：对步兵(防8)约38伤，对侦察兵(防4)约42伤

步兵28攻：对步兵(防8)约20伤
```

### 3. 评分机制（重要！）
```
总分 = 敌HQ伤害×4 + 我方HQ HP×2 + 据点×120 + 存活兵力价值×2 + 剩余补给×1
本局比分: (57×4=228) + (200×2=400) + (3×120=360) + (663×2=1326) + 185 = 2499
vs 敌方: (0×4=0) + (143×2=286) + (1×120=120) + (138×2=276) + 85 = 767
```
> **结论：** 存活兵力价值(1326分)是最大得分项！保全兵力比囤积补给更重要
