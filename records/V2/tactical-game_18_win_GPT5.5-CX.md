## 核心胜利策略

### 1. 三回合控三点，第四回合夺北点

**关键决策:** 前三回合不追求击杀，而是先拿cp_b、cp_c、cp_s，并用第4回合scout夺cp_n。

```
Turn 1: infantry(7,0)->cp_b(4,0)
Turn 2: scout(2,1)->cp_c(0,0)
Turn 3: scout(3,1)->cp_s(0,4)
Turn 4: scout(2,-1)->cp_n(0,-4)
```

**为什么有效:**
- 第4回合结束后B控制4点，之后连续获得70补给/回合。
- A虽然第1-2回合拿到cp_a和cp_n，但cp_n在第4回合被抢回，经济优势开始倒向B。
- 三点以上收入保证了heavy、support、ranger都能按节奏上线。

### 2. Heavy压中央，ranger负责收割

**关键决策:** 第4回合花90补给在(1,0)部署heavy，让它进入(0,0)吸收火力，ranger和步兵负责击杀。

```
Turn 4: deploy heavy(1,0)
Turn 5: heavy(0,0)攻击A infantry到49HP
Turn 6: heavy再击A infantry到18HP，ranger补刀击杀
```

**为什么有效:**
- heavy 145HP/14防，A方投入ranger+步兵也只能把它打到103HP，无法快速击杀。
- 中央heavy逼A消耗行动点处理正面，给两翼scout夺点创造空间。
- 这直接印证了第17局经验：heavy不是唯一输出点，但能稳定建立正面压力。

### 3. 控全图后再转HQ，斩杀更稳

**关键决策:** 第7回合先夺cp_a并清support，不急着打HQ；第8回合才开始以ranger群攻击总部。

```
Turn 7: infantry(-4,0)夺cp_a，B全控5点
Turn 8: 新ranger(-5,0)打HQ 200->165
Turn 9: ranger+ranger+ranger+scout打HQ 165->40
Turn 10: 双ranger打HQ 40->0
```

**为什么有效:**
- 全控5点后每轮收入85，A方只剩基础10，无法补出有效反制。
- ranger射程3可以从(-5,0)、(-5,-1)、(-6,0)等格子安全攻击HQ。
- 第10回合只需要2个行动点就能结束，剩余行动点不用暴露给A反扑。

### 4. 充分利用“已激活单位免费后续动作”

**关键决策:** 多次让同一单位先攻击再移动，或部署后直接攻击，把5行动点用出超过5个动作的效果。

```
Turn 6: scout攻击A ranger后移动/推进位置
Turn 7: infantry击杀后免费移动到cp_a
Turn 9: ranger攻击HQ后移动到(-6,0)，腾出(-5,0)再部署新ranger
```

**为什么有效:**
- 一次激活只花1行动点，同单位后续移动/攻击不再增加actionsUsed。
- 这让B在第7-9回合可以同时完成击杀、夺点、部署和HQ输出。
- V2里行动点比补给更稀缺，免费后续动作是扩大优势的核心机制。

## 失误与改进

### 失误1: 第9回合误判地形，尝试走到阻挡格

**问题:** 曾计划让ranger从(-3,-2)走到(-5,-2)攻击HQ，但(-5,-2)是blocker，导致API拒绝；虽然没有消耗行动点，但浪费了实时决策时间。
**改进:** 进入HQ区前先检查关键地形：(-5,-2)是blocker，(2,0)也是blocker，南北水域在r=±2附近会影响路径。
**收益:** 终局斩杀阶段减少无效尝试，避免在限时或同步对局里错失最佳动作。

### 失误2: 第3回合没有直接击杀A scout

**问题:** 第3回合只把A scout(0,-1)压到33HP，没有击杀；它后来参与A方反击并造成我方中心单位压力。
**改进:** 若中心争夺已经确定，优先用ranger或额外scout补刀低血捕点单位。
**收益:** 更早消灭A的机动夺点单位，降低cp_n和cp_c被反夺风险。

### 失误3: 支援部署偏晚，导致中心步兵阵亡

**问题:** support第5回合才部署，我方中心步兵在第5回合被A击杀。虽然整体优势未受影响，但这是本局唯一单位损失。
**改进:** 如果第4回合选择不立刻heavy，可以改为support+保守站位；或第5回合优先治疗中心步兵而非前插步兵。
**收益:** 保住一个100HP步兵，终局可多一个贴HQ或守点单位。

### 失误4: 第8回合偏向清场，HQ斩杀延后一回合

**问题:** 第8回合用多次行动击杀A infantry，而不是最大化HQ输出；结果HQ在第9回合后仍剩40HP，需要第10回合补刀。
**改进:** 如果判断A已无足够反杀能力，第8回合可以多部署/移动ranger进入HQ射程，提前一回合结束。
**收益:** 有机会把胜利从第10回合压缩到第9回合。

## 游戏机制深度理解

### 1. 经济模型

```
起始补给: 80
基础收入: 10/回合
控制点收入: 15/点/回合
五点全控收入: 10 + 15*5 = 85/回合

本局B方:
  收入合计: 595
  部署支出: 565
  最终补给: 110
```

结论：第7回合全控5点后，A只有基础10收入，B每轮85收入。经济差足以支持连续部署ranger终结HQ。

### 2. 攻击机制

```
伤害 = max(1, 攻击力 - 防御力 + random(-3, 3))
ranger 46攻打HQ 8防: 约35-41伤
scout 18攻打HQ 8防: 约7-13伤
heavy 36攻打步兵8防: 约25-31伤
```

结论：打HQ的主力必须是ranger。scout贴脸可补伤害，但不能替代ranger。

### 3. 部署后可攻击

```
deploy 会让新单位 hasMoved=true、actionSpent=true
但 hasActed=false，因此若目标在射程内，部署后可以直接攻击
```

本局第8、9回合多次从cp_a部署ranger到(-5,0)/(-5,1)，当回合直接射击HQ或A scout，这是终局加速的关键。

### 4. 行动点上限

```
每回合最多激活5个新单位
已激活单位后续移动/攻击/治疗不再额外消耗行动点
```

结论：强回合应该优先激活“能同时移动+攻击”的单位；单纯移动但没有攻击价值的单位要让位给部署ranger或击杀关键目标。

## 总结

### 胜利关键因素

1. 第1-4回合快速取得cp_b/cp_c/cp_s/cp_n，提前建立4点收入。
2. 第4回合heavy上线，迫使A方无法靠步兵/ranger快速突破中央。
3. 第7回合夺cp_a，形成5点全控，A经济降到基础10。
4. 第8-10回合连续部署ranger并集中攻击HQ，最终第10回合摧毁总部。
5. 对行动点机制利用充分，多次做到移动+攻击、部署+攻击、攻击后移动。

### 核心战术原则

> “四回合控经济，七回合全图，八回合转HQ。”

- scout负责抢点和补刀，不要长期正面抗线。
- heavy负责中线压制，核心价值是吸收火力。
- ranger是HQ终结单位，进入射程后优先打总部。
- support要贴近主力，否则治疗射程1很难发挥。
- 终局前必须检查blocker和水域，避免无效移动。

### 一句话总结

**本局胜利来自控制点经济、heavy中线压制和ranger群HQ斩杀三段衔接；只要第7回合形成五点全控，第8回合后就应该把所有补给和行动点转化为射程3总部伤害。**
