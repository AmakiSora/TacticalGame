## 核心胜利策略

### 1. 快速占领控制点建立经济优势

**关键决策:** 第1回合立即派遣scout占领东侧补给站(cp_b)，第3回合派遣另一个scout占领南部中继站(cp_s)

```
Turn 1: scout从(7,1)移动到(4,0)占领cp_b — 获得15金/回合收入
Turn 3: scout从(1,3)移动到(0,4)占领cp_s — 获得15金/回合收入
```

**为什么有效:**
- 两个控制点提供30金/回合额外收入
- 虽然中期cp_s被敌人夺回，但cp_b始终保持
- 经济优势为后期大量部署步兵提供基础

### 2. 持续步兵海战术

**关键决策:** 全程只部署infantry（45金），不部署昂贵的heavy（90金）或ranger（75金）

```
Turn 1: deploy infantry(7,1), deploy scout(8,-1)
Turn 3: deploy infantry(3,1)
Turn 4: deploy infantry(1,3)
Turn 5: deploy infantry(2,-1)
...
Turn 20: 总共部署约12个infantry
```

**为什么有效:**
- infantry性价比最高：100HP、28攻击、45金
- 数量优势形成压倒性火力
- 每回合5个行动点可以控制5个单位攻击
- 从第17局失败中学到：不能只用scout，需要infantry作为主力

### 3. 集火攻击消灭敌人关键单位

**关键决策:** 优先攻击敌人的support和ranger，防止敌人治疗和远程打击

```
Turn 4-8: 连续攻击敌人ranger(0,1)和support(1,0)
Turn 6: 两个infantry集火support(1,0)
Turn 8: 成功消灭ranger(0,1)
```

**为什么有效:**
- support可以治疗20HP/次，必须优先消灭
- ranger攻击46点，威胁最大
- 集火攻击确保杀死敌人，避免浪费行动点

### 4. 保护残血单位撤退

**关键决策:** 当单位HP较低时，及时撤退到安全位置

```
Turn 4: scout从(0,4)撤退到(1,4)，避免被敌人攻击
Turn 11: infantry从(3,-2)撤退到(4,-1)
```

**为什么有效:**
- 保存有生力量，避免白白损失
- 残血单位可以作为后续攻击的补充
- 从第17局失败中学到：不能让残血单位硬拼

### 5. 灵活调整攻击目标

**关键决策:** 根据战场形势，灵活选择攻击目标

```
Turn 1-5: 优先占领控制点
Turn 6-10: 集火攻击敌人ranger和support
Turn 11-15: 攻击敌人东侧部队
Turn 16-20: 消灭敌人残余，最大化得分
```

**为什么有效:**
- 不固执于单一目标，根据形势调整
- 确保每个行动点都产生最大价值
- 最终以军队价值优势赢得裁判判决

## 失误与改进

### 失误1: 中期丢失南部中继站

**问题:** 第7回合敌人scout(0,4)夺走了我的cp_s，导致收入从55降到40金/回合

**改进:** 应该在cp_s附近保留一个infantry防守，而不是全部推到前线

**收益:** 如果保持cp_s，总收入会多15金/回合×13回合=195金，可以多部署4个infantry

### 失误2: 部分攻击浪费行动点

**问题:** 第5回合攻击敌人infantry(-1,0)但没有杀死，第6回合继续攻击才杀死

**改进:** 应该先计算伤害，确保一次攻击能杀死再行动

**收益:** 节省1个行动点，可以用于其他更有价值的操作

### 失误3: 部署位置不够优化

**问题:** 多次尝试从cp_b部署到(5,1)但失败，因为不相邻

**改进:** 提前规划部署位置，确保在控制点相邻格子有空位

**收益:** 避免浪费时间尝试无效操作，提高行动效率

### 失误4: 没有尝试攻击敌人HQ

**问题:** 全程没有尝试攻击敌人HQ(-8,0)，错失直接胜利机会

**改进:** 从第15回合开始，应该派遣部队向敌人HQ推进

**收益:** 如果成功摧毁HQ，可以获得更早的胜利，得分更高

## 游戏机制深度理解

### 1. 经济模型

```
基础收入: 10金/回合
控制点收入: 15金/回合/个
本局收入:
- 第1-3回合: 10+15=25金/回合 (1个cp_b)
- 第3-7回合: 10+30=40金/回合 (cp_b+cp_s)
- 第7-20回合: 10+15=25金/回合 (只有cp_b)
```

### 2. 单位性价比

```
infantry: 45金, 100HP, 28攻击 → 性价比最高
scout: 40金, 70HP, 18攻击 → 适合侦察和占领
heavy: 90金, 145HP, 36攻击 → 太贵，不划算
ranger: 75金, 75HP, 46攻击 → 远程但太脆
support: 60金, 80HP, 12攻击, 20治疗 → 辅助单位
```

### 3. 裁判判决得分公式

```
得分 = 敌方HQ受损×4 + 己方HQ HP×2 + 控制点×120 + 军队价值×2 + 补给×1

本局最终得分:
- 我: 0×4 + 200×2 + 2×120 + 523×2 + 45×1 = 0+400+240+1046+45 = 1731
- 对手: 0×4 + 200×2 + 3×120 + 150×2 + 625×1 = 0+400+360+300+625 = 1685
```

> 军队价值差距523 vs 150是胜利关键，说明步兵海战术有效
