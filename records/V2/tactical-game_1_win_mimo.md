## 核心胜利策略

### 1. 据点控制战略

**关键决策:** 优先占领所有5个据点，建立经济优势

```
Turn 4: 夺回中央阵地(cp_c) — 占领中立据点
Turn 6: 占领南部中继站(cp_s) — 扩大控制范围
Turn 10: 夺取东侧补给站(cp_b) — 完全控制所有据点
```

**为什么有效:**
- 每个据点提供20金/回合收入
- 5个据点 = 100金/回合额外收入
- 基础收入15金 + 100金 = 115金/回合
- 经济优势碾压对手（对手仅15金/回合）

### 2. 混合部队编成

**关键决策:** 建立多元化的部队组合

```
Turn 1-3: 步兵+侦察兵+重装兵 — 基础战斗力量
Turn 5-6: 重装兵+侦察兵 — 增强前线
Turn 7-8: 2个游骑兵 — 远程火力支援
Turn 9-10: 2个支援兵 — 治疗保障
Turn 11-16: 6个步兵 — 占领和推进
```

**为什么有效:**
- 步兵：占领据点，推进战线
- 侦察兵：快速移动，占领据点
- 重装兵：高HP高防御，前线坦克
- 游骑兵：远程攻击，安全输出
- 支援兵：治疗伤员，维持战斗力

### 3. 集中火力战术

**关键决策:** 集中多个单位攻击同一目标

```
Turn 3: 3个单位集中攻击敌方侦察兵 — 击杀
Turn 6: 3个单位集中攻击敌方游骑兵 — 击杀
Turn 9: 5个单位集中攻击敌方重装兵 — 击杀
Turn 10: 4个单位集中攻击敌方支援兵 — 击杀
Turn 16: 4个单位集中攻击敌方HQ — 摧毁
```

**为什么有效:**
- 快速击杀敌方单位，减少我方损失
- 避免分散火力，让敌方单位存活更久
- 优先击杀高威胁目标（游骑兵、支援兵）

### 4. 前线推进策略

**关键决策:** 稳步向敌方HQ推进

```
Turn 1-5: 建立前线据点
Turn 6-10: 控制中央区域
Turn 11-15: 向敌方HQ推进
Turn 16: 最终摧毁
```

**为什么有效:**
- 控制关键位置，限制敌方活动空间
- 逐步压缩敌方生存空间
- 最终集中兵力摧毁HQ

## 失误与改进

### 失误1: 早期侦察兵损失

**问题:** 第5回合侦察兵被击杀
**改进:** 侦察兵应该更谨慎，避免进入敌方攻击范围
**收益:** 保留侦察兵可以更快占领据点

### 失误2: 部分单位闲置

**问题:** 部分单位没有有效行动
**改进:** 每回合确保所有单位都有行动（移动或攻击）
**收益:** 提高整体效率

### 失误3: 治疗时机

**问题:** 支援兵治疗时机不够及时
**改进:** 优先治疗低HP单位，保持部队战斗力
**收益:** 减少单位损失

## 游戏机制深度理解

### 1. 经济模型
```
基础收入: 15金/回合
据点收入: 20金/回合/个
总收入: 15 + 20×控制据点数
```

### 2. 攻击机制
```
伤害 = max(1, 攻击力 - 防御力 + random(-3, 3))
例: 游骑兵(42攻) vs 侦察兵(4防) = 38±3 = 35-41伤害
```

### 3. 据点占领
```
占领条件: 步兵或侦察兵站在据点上，回合结束时占领
占领效果: 获得该据点收入
```
