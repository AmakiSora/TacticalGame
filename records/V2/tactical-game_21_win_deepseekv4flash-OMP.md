## 核心胜利策略

### 1. 闪电双占点 — 第1回合夺取cp_b，第3回合夺取cp_c

**关键决策:** 首回合直接步兵占领东侧补给站，第3回合合围击杀中心侦察兵后占领中央阵地。不在开局部署无意义单位。

```
Turn 1: 步兵(7,0)→(4,0)占领cp_b — 立即建立15金/回合额外收入
Turn 3: 双步兵夹击→侦察兵补刀击杀中央侦察兵→(0,0)占领cp_c
        — 代价：仅1行动点 + 3次免费攻击
```

**为什么有效:**
- 第3回合就控制2据点 = 40金/回合收入，远超敌方10金基础
- 控制了中央(0,0)就控制了地图核心，分割敌军南北联系
- 为后续cp_s的占领和3据点经济奠定基础

### 2. 游骑兵远程压制 + 步兵包抄的击杀体系

**关键决策:** 游骑兵（46攻击/3射程）承担主要输出，步兵包抄到近战位补刀。

```
常规击杀流程:
Step 1: 游骑兵从3格外狙击 — 造成35-45伤害（步兵）或40-46伤害（侦察兵）
Step 2: 步兵移动到相邻位 — 免费近战补刀20-26伤害
Step 3: 如未死, 另一个步兵或侦察兵继续补刀
```

**为什么有效:**
- 游骑兵46攻击是游戏最高单体伤害，且3格射程让敌人无法反击
- 步兵只负责最后一击，减少近战单位暴露在敌人火力下的时间
- 击杀节奏极快：第3回合首杀 → 最终击杀10+个敌方单位

### 3. 重坦中盘入场 — 战场主宰

**关键决策:** 第6回合（补给100时）部署第一辆重坦(145HP/36攻)，第8回合部署支援。

```
Turn 6: 部署重坦(7,0)花90 — 补给100→10
Turn 7-9: 重坦(7,0)→(5,0)→(3,0)— 绕过(2,0)路障从北线推进
Turn 10+: 重坦(1,0)→(-1,0)→(-2,1)→(-3,1)
          — 与游骑兵配合作战
```

**为什么有效:**
- 145HP可以抗住几乎所有敌人的攻击，成为前线"盾牌"
- 36攻击对敌方重坦造成22伤，对轻甲单位造成28-32伤
- 重坦压制迫使敌方将火力集中在它身上，保护游骑兵和步兵

### 4. 地形利用 — 路障和水域的攻防影响

**关键决策:** 理解(-2,0)和(2,0)路障以及(-1~1, ±2)水域对路径的影响。

```
路障影响:
- 赤道路障(±2,0)迫使进攻必须绕北或绕南
- 我方选择南线推进：(1,-1)→(0,-1)→(-1,0)→(-2,1)→(-3,1)
- 北部水域(-1~1,2)阻止直接通往cp_s或cp_n
```

**为什么有效:**
- 南线绕路经过(0,-1)→(-1,0)→(-2,1)是最短通往敌方阵地的路径
- 正确理解地形让我能在规划和执行时节省大量行动点
- 水域+路障形成天然防线，我利用它们预判敌方可能的进攻路线

## 失误与改进

### 失误1: 单位相互阻挡，推进效率低

**问题:** 是我HQ位于东侧、重坦和步兵从中心向西推时，步兵(-1,1)经常阻挡重坦(-2,1)的去路，导致重坦需要多花一回合绕路。
- 第10回合: 重坦(0,1)想去(-2,1)被步兵(-1,1)挡住，只能绕到(1,0)→(-1,0)
- 第14回合: 重坦从(-1,0)到(-2,1)又因为步兵挡路无法推进

**改进:** 推进时应保持"快速单位打头阵、慢速单位跟随后"的纵队顺序，近战步兵在前挡路时应主动让出一条路让重坦通过。

**收益:** 每局至少节省1-2个行动点（相当于多部署一个侦察兵或一次多一次战术行动）

### 失误2: 游骑兵远程部署位置不够靠前

**问题:** 第6回合部署的新游骑兵(3,1)需要走3回合才进入有效射程，错过了中盘关键输出窗口。

**改进:** 从cp_b(4,0)部署游骑兵时应选(3,1)或(4,1)等更靠近战场的位置，或者直接从HQ部署到(7,1)再通过两步移动到(5,1)节省推进时间。

**收益:** 游骑兵可以提前2回合进入战场，多输出2-3次狙击(每个造成35-46伤)

### 失误3: 未控制北部中继站cp_n

**问题:** 北部中继站(0,-4)整局都在敌方控制下，虽然我的侦察兵多次击杀了站岗的敌人，但始终没有派自己的单位上去占领。

**改进:** 第8回合击杀cp_n敌方侦察兵后，应该立即派一个侦察兵或步兵绕过水域从东侧(2,-2)→(1,-3)→(0,-4)占领。

**收益:** 多一个据点=120评分+15金/回合，最终比分差距会从947扩大到1067
