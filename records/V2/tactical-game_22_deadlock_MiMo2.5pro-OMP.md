## 核心胜利策略

### 1. 侦察兵闪电占CP

**关键决策:** 开局用侦察兵(移动力5)快速抢占中央和北部CP

```
Turn 1: 侦察兵→(-2,-1) 快速推进
Turn 2: 侦察兵→(0,0) 占领中央cp_c
Turn 20: 侦察兵→(0,-4) 占领北部cp_n — 决定性一击！
```

**为什么有效:**
- 侦察兵移动力5，3回合内可从HQ到达任何CP
- 早期占CP获得持续经济优势(每个CP+15/回合)
- 最后一回合抢占cp_n，CP数3:2，权重120，直接决定胜负

### 2. 重装步兵前线碾压

**关键决策:** 第9回合开始部署重装步兵(HP:145, 攻击:36, 防御:14)

```
Turn 9: 部署重装步兵→(-7,0)，花费90补给
Turn 13: 部署第二个重装步兵→(-7,0)，花费90补给
```

**为什么有效:**
- 重装步兵HP:145，是步兵的1.45倍
- 攻击36，防御14，单次伤害高达22-35
- 2个重装步兵形成钢铁洪流，敌方步兵无法阻挡

### 3. 集火战术 — 多单位围攻单点

**关键决策:** 每回合集中2-3个单位攻击同一个敌方目标

```
Turn 7: 步兵1攻击(0,0)+步兵2攻击(0,0)+步兵3攻击(0,0) = 三路夹击
Turn 12: 步兵(0,-1)击杀(0,0)+步兵(1,0)攻击(1,1)+步兵(0,1)攻击(1,1)
```

**为什么有效:**
- 步兵攻击28，3次攻击=84伤害，足以击杀大部分单位
- 集火可以快速减员，形成局部人数优势
- 避免分散攻击导致敌方单位都能反击

### 4. 经济管理 — 平衡部署与推进

**关键决策:** 前期占CP积累经济，中期部署重装步兵，后期保持补给

```
Turn 1-2: 占2个CP，收入40/回合
Turn 9: 100补给，部署重装步兵(-7,0)花费90
Turn 13: 150补给，部署第二个重装步兵花费90
Turn 20: 235补给，最后一搏部署+推进
```

**为什么有效:**
- 早期占CP提供持续收入(40/回合 vs 对手25/回合)
- 重装步兵90补给虽然昂贵，但性价比极高
- 保持100+补给储备应对突发情况

### 5. 伤兵管理 — 及时撤退残血单位

**关键决策:** 每回合撤退HP<30的单位到后方

```
Turn 11: 撤退3HP步兵(-1,1)→(-4,1)
Turn 14: 撤退3HP步兵(-4,1)→(-7,1)
Turn 17: 撤退30HP侦察兵(0,0)→(-2,-1)
Turn 18: 撤退23HP步兵(0,1)→(-1,1)
```

**为什么有效:**
- 残血单位留在前线会被轻易击杀，浪费行动点
- 撤退到后方可以保全部队价值(45-40补给)
- 后方单位轮换上前线，保持持续战斗力

## 失误与改进

### 失误1: 中央阵地反复易手

**问题:** 第2回合占领中央cp_c后，第4-5回合被敌方夺回，反复争夺3次
**改进:** 占领中央后应立即部署防御单位，而不是继续推进
**收益:** 避免浪费行动点在反复争夺上，可以更早推进到敌方领地

### 失误2: 重装步兵部署过晚

**问题:** 直到第9回合才部署第一个重装步兵，前期只有步兵和侦察兵
**改进:** 第5-6回合经济允许时就应该部署重装步兵(90补给)
**收益:** 重装步兵提前3回合上线，可以更早形成碾压优势

### 失误3: 后方单位推进缓慢

**问题:** 后方步兵(-8,1)等单位多次尝试推进失败(路径被堵)
**改进:** 前线单位推进时应主动为后方单位让路
**收益:** 避免行动点浪费，后方单位可以更快投入战斗

### 失误4: 游侠威胁反应过慢

**问题:** 第13-17回合游侠(3,1)和(3,-1)持续输出，直到第19回合才全部击杀
**改进:** 游侠射程3，应在进入其射程前集火击杀
**收益:** 减少我方单位损失(多个步兵被打到残血)
