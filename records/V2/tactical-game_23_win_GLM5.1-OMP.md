## 核心胜利策略

### 1. 闪电三据点 — 第1回合cp_b、第2回合cp_c、第5回合cp_s

**关键决策:** 前5回合凭借初始单位的机动性快速占领三个控制点，建立经济碾压基础。

```
Turn 1: 侦察兵(7,1)→(4,0)占领cp_b — 立即增加15金/回合
Turn 2: 侦察兵(4,0)→(0,0)占领cp_c — 中央阵地控制
Turn 5: 侦察兵(1,3)→(0,4)占领cp_s — 三据点锁定
```

**为什么有效:**
- 第5回合就控制3据点 = 55金/回合收入（10基础+45据点），碾压敌方25金/回合
- 控制了中央(0,0)后，南北两侧控制点(0,4)和(0,-4)成为下一阶段的天然目标
- 经济优势让后期每回合都能部署1-2个新单位

### 2. 游骑兵远程集火链 — 安全击杀体系

**关键决策:** 用游骑兵(46攻/3射程)从安全距离远程狙击，已激活的游骑兵再配合步兵近战补刀，实现"无伤击杀"。

```
常规击杀流程:
Step 1: 游骑兵从3格外远程狙击 — 对步兵≈38伤，对侦察兵≈42伤
Step 2: 同游骑兵继续输出（已激活，免费）或切换游骑兵轮换
Step 3: 多个游骑兵轮换射 → 重坦只需3次狙击即可击杀(46-14+随机≈32/次，145HP→4-5次)
```

**为什么有效:**
- 3格射程意味着大多数敌人在死前都无法反击
- 游骑兵的46攻击是游戏最高单体伤害，性价比极高(75金)
- 多个游骑兵可以轮换输出，5行动点内可以完成3次狙击+2次部署

### 3. 支援单位中盘入场 — 前线续航保障

**关键决策:** 第6回合部署支援(60金)，第7回合部署重型(90金)，形成"重坦扛线+支援回血+游骑兵输出"的黄金三角。

```
Turn 6: 部署支援(7,1)花60金
Turn 7-12: 支援从(7,1)→(5,1)→(3,1)→(1,1)→(0,1)→(-1,0)
           逐步推进至前线，沿途治疗受伤单位
```

**为什么有效:**
- 支援每回合治疗20HP，显著延长前排单位的战场存活时间
- 游骑兵(0,-1)被敌方反击到19HP后，支援及时治疗恢复到59HP
- 支援本身80HP/5防足够在第二线安全存活

### 4. 地形规避 — 水域和路障的战略影响

**关键决策:** 理解点顶六边形的赤道路障(±2,0)和水域(0,±2)对路径的约束，选择最优推进路线。

```
水障影响(0,±2):
- 南部(0,4)到中央(0,0)被(0,2)水域阻断
- 绕过路径：(0,4)→(1,3)→(1,1)→(0,0) 或 (0,4)→(-1,3)→(-1,1)→(0,0)

路障影响(±2,0):
- 从中央到敌方HQ必须绕北(-2,1)或绕南(-2,-1)
- (-2,0)阻断直线推进
```

**为什么有效:**
- 正确理解绕路路径，使侦察兵能顺利从中央到达南部中继站
- 步兵从(3,-2)推进到北部中继站时选择了(1,-3)→(0,-3)的绕水路线

## 失误与改进

### 失误1: 始终未占领北部中继站cp_n(0,-4)

**问题:** 第5回合击杀cp_n敌方侦察兵后，北部中继站处于无人控制状态。但后续3-4个回合我都没有派单位去占领，被敌方重新部署侦察兵抢回。

**改进:** 击杀cp_n守军后应立即派一个步兵或侦察兵从东侧(1,-3)→(0,-4)过去占领。第5-6回合我有足够的行动点和空闲单位执行此操作。

**收益:** 多一个据点=120评分+15金/回合，比分差距可扩大约150分

### 失误2: 前线扎堆，单位互相阻挡

**问题:** 第10-13回合游骑兵、步兵、重型、支援都挤在中央(-1,0)到(1,1)区域，互相阻挡导致重型无法向前推进，有一回合重型被迫后退(1,0)让路。

**改进:** 推进时应保持梯队：游骑兵在最前方输出，步兵在中路掩护，重型和支援在后面跟进。发现阻挡时，让路单位主动让出关键路径。

**收益:** 减少1-2回合的无效移动，相当于多出2次攻击机会

### 失误3: 前期游骑兵推进不够激进

**问题:** 首回合部署的游骑兵(8,-1)在第2-4回合移动较为保守(6,-1)→(4,-1)→(4,-1)，没有尽快进入攻击位置。

**改进:** 游骑兵应尽快推进到(2,-1)或(1,-1)附近，进入射程覆盖敌方前线的位置。

**收益:** 游骑兵可以提前1-2回合进入战斗，多输出2次狙击(每个≈40伤)


```
伤害公式: max(1, 攻击力 - 防御力 + random(-3, 3))
游骑兵46攻: 对步兵(防8)≈38伤, 对侦察兵(防4)≈42伤, 对重坦(防14)≈32伤

侦察兵18攻: 对步兵≈10伤, 对侦察兵≈14伤
```

### 3. 评分机制
```
总分 = 敌HQ伤害×4 + 我方HQ HP×2 + 据点×120 + 存活兵力×2 + 剩余补给×1
```

### 4. 部署规则
```
部署来源: 己方HQ或已占领的控制点
部署条件: 部署点相邻的空地(plain)，且目标不在任何单位/路障/水域上
部署单位: 同一回合部署的单元已激活，不能再行动
```

### 5. 地形关键
```
路障(不可通行): (-2,0), (2,0), (-5,-2), (5,2)
水域(不可通行): (0,±2), (-1,±2), (1,±2)
```

1. **⚡ 闪电三据点（第1、2、5回合）** — 奠定经济碾压基础

- 推进时保持梯队，避免单位互相阻挡浪费行动点
- 已占控制点旁的空地是理想的前线部署点

### 一句话总结

**第23局以3据点经济和游骑兵远程集火体系稳固取胜，验证了在Hex V2中"经济先于一切、远程优于近战"的战术理念，但未趁势占领北部中继站是本局最大的战术遗憾。**
