## 游戏机制深度理解

### 1. 经济模型（六角前线 v2）
```
基础收入: 10金/回合
CP收入: 12-15金/CP/回合（本局cp_a/b为12金，cp_c/n/s为12金）
4CP对手: 10+48 = 58金/回合
1CP我: 10+12 = 22金/回合
差距: 36金/回合

Turn 5-15(11回合): 对手额外获得 36×11 = 396金
```

### 2. 六角网格距离公式
```
dist((q1,r1), (q2,r2)) = max(|dq|, |dr|, |ds|)
其中 s = -q-r, ds = |s1-s2|
示例: (7,0)→(0,0): dq=7, dr=0, ds=7, dist=7
      (8,0)→(-8,0): dq=16, dr=0, ds=16, dist=16
      (2,1)→(1,1): dq=1, dr=0, ds=1, dist=1
      (1,-1)→(0,-1): dq=1, dr=0, ds=1, dist=1
      (1,-1)→(0,0): dq=1, dr=1, ds=2, dist=2
```

### 3. 攻击机制
```
伤害 = max(1, 攻击力 - 防御力 + random(-3, 3))
infantry攻击: 28
scout攻击: 18
infantry防御: 5
敌方infantry防御: 5
infantry→infantry: max(1, 28-5+random(-3,3)) = 20~28/次
scout→infantry: max(1, 18-5+random(-3,3)) = 10~18/次
scout→scout: max(1, 18-5+random(-3,3)) = 10~18/次
```

### 4. 行动点限制
```
每回合5个行动点
部署=1AP，移动=1AP，攻击=1AP
已激活单位在当回合内可免费移动（actionsRemaining>0）
最优策略：先攻击（消耗AP），再免费移动到新位置
```

## 总结

### 失败关键因素
1. **经济碾压** — 对手4CP vs 我1CP，每回合36金差距，15回合累积396金差距
2. **Player B的先天劣势** — 从右侧(8,0)到中央CP的距离远于对手从左侧(-8,0)到cp_a的距离
3. **兵力差距** — 对手部署15个单位 vs 我9个，步兵5:10的劣势导致正面战场被压制
4. **缺乏HQ威胁** — 0点HQ伤害意味着在分数判定时完全缺乏"有效输出"

### 核心战术原则

> **"单CP在20回合游戏中必败——必须双CP起步，否则经济永远追不上"**

- 作为Player B，必须在Turn 3前抢cp_c(0,0)，否则经济差距不可逆转
- 部署infantry优先于scout，前线需要步兵突破
- 集中4-5个infantry在一点突破，而不是分散6个scout
- HQ伤害在分数判定中是重要指标，即使0 vs 0也要尽量施加压力

### 一句话总结

**"本局失败的根本原因是对手前4回合闪电占领4个CP形成的经济碾压——我每回合净亏损36金，从Turn 5起就注定败局；作为Player B从右侧起步，必须更激进地抢CP才能扭转先天劣势。"**
