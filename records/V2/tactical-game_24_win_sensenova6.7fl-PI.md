## 核心胜利策略

### 1. 闪电四据点 — 前4回合全占

**关键决策:** 前4回合凭借初始单位的机动性快速占领全部4个中立控制点，建立经济碾压基础。

```
Turn 1: 步兵(-7,0)→(-4,0)占领cp_a — 立即增加12金/回合
Turn 2: 侦察兵(-2,-3)→(0,-4)占领cp_n — 北部中继站控制
Turn 3: 步兵(-1,0)→(0,0)占领cp_c — 中央阵地控制
Turn 4: 侦察兵(0,1)→(0,4)占领cp_s — 南部中继站控制
```

**为什么有效:**
- 第4回合就控制4据点 = 58金/回合收入（10基础+48据点），碾压敌方22金/回合
- 控制了中央(0,0)后，南北两侧控制点(0,4)和(0,-4)成为下一阶段的天然目标
- 经济优势让后期每回合都能部署1-2个新单位
- 敌方仅控制cp_b(4,0)一个据点，经济差距悬殊

### 2. 步兵群推进 — 持续消耗敌方前线

**关键决策:** 利用经济优势持续部署步兵(45金)，形成步兵群推进，逐步消耗敌方前线单位。

```
常规推进流程:
Step 1: 从cp_c(0,0)或cp_a(-4,0)部署步兵到前线
Step 2: 步兵向敌方前线推进，逐步压缩敌方空间
Step 3: 多个步兵集火攻击敌方单位，逐步消耗
```

**为什么有效:**
- 步兵45金性价比适中，100HP/8防御足够在前线存活
- 步兵28攻击力对侦察兵(防4)≈24伤，对步兵(防8)≈20伤
- 多个步兵可以轮换攻击，5行动点内可以完成3次攻击+2次部署

### 3. 侦察兵远程狙击 — 安全削血体系

**关键决策:** 用侦察兵(18攻/1射程)从安全距离远程狙击，已激活的侦察兵再配合步兵近战补刀，实现"无伤击杀"。

```
常规击杀流程:
Step 1: 侦察兵从1格外远程狙击 — 对步兵≈10伤，对侦察兵≈14伤
Step 2: 同侦察兵继续输出（已激活，免费）或切换侦察兵轮换
Step 3: 多个侦察兵轮换射 → 敌方单位只需3-4次狙击即可击杀
```

**为什么有效:**
- 1格射程意味着大多数敌人在死前都无法反击
- 侦察兵18攻击是游戏最低单体伤害，但胜在安全
- 多个侦察兵可以轮换输出，5行动点内可以完成3次狙击+2次部署

### 4. 地形规避 — 水域和路障的战略影响

**关键决策:** 理解点顶六边形的赤道路障(±2,0)和水域(0,±2)对路径的约束，选择最优推进路线。

```
水障影响(0,±2):
- 南部(0,4)到中央(0,0)被(0,2)水域阻断
- 绕过路径：(0,4)→(1,3)→(1,1)→(0,0) 或 (0,4)→(-1,3)→(-1,1)→(0,0)

路障影响(±2,0):
- 从中央到敌方HQ必须绕北(-2,1)或绕南(-2,-1)
- (-2,0)阻断直线推进
```

**为什么有效:**
- 正确理解绕路路径，使侦察兵能顺利从中央到达南部中继站
- 步兵从(3,-2)推进到北部中继站时选择了(1,-3)→(0,-3)的绕水路线

## 失误与改进

### 失误1: 侦察兵远程狙击效率低

**问题:** 侦察兵18攻击力对步兵(防8)≈10伤，对侦察兵(防4)≈14伤，击杀效率远低于步兵28攻击力。第5-7回合侦察兵(1,-1)连续3回合攻击敌方侦察兵(2,-1)，仅造成48伤(18+15+15)，而步兵28攻击力只需2次攻击即可击杀。

**改进:** 优先部署步兵而非侦察兵，步兵28攻击力对步兵(防8)≈20伤，对侦察兵(防4)≈24伤，击杀效率更高。

**收益:** 减少2-3回合的无效攻击，相当于多出2-3次攻击机会

### 失误2: 前线扎堆，单位互相阻挡

**问题:** 第10-14回合步兵、侦察兵、支援都挤在中央(-1,0)到(1,1)区域，互相阻挡导致步兵无法向前推进，有一回合步兵被迫后退(1,0)让路。

**改进:** 推进时应保持梯队：步兵在最前方输出，侦察兵在中路掩护，支援在后面跟进。发现阻挡时，让路单位主动让出关键路径。

**收益:** 减少1-2回合的无效移动，相当于多出2次攻击机会

### 失误3: 前期侦察兵推进不够激进

**问题:** 首回合部署的侦察兵(-7,-1)在第2-4回合移动较为保守(-2,-3)→(0,-4)→(0,-4)，没有尽快进入攻击位置。

**改进:** 侦察兵应尽快推进到(2,-1)或(1,-1)附近，进入射程覆盖敌方前线的位置。

**收益:** 侦察兵可以提前1-2回合进入战斗，多输出2次狙击(每个≈14伤)

## 游戏机制深度理解（Hex V2）

### 1. 经济模型
```
基础收入: 10金币/回合（每玩家）
控制点收入: 12金币/据点/回合
行动点: 5行动点/回合（激活单位用）
起始补给: 80
本局收入变化:
  第1回合: 10（仅基础）
  第2回合: 22（cp_a）
  第3回合: 34（cp_a+cp_n）
  第4回合: 46（cp_a+cp_n+cp_c）
  第5-15回合: 58（cp_a+cp_n+cp_c+cp_s）
```

### 2. 攻击机制
```
伤害公式: max(1, 攻击力 - 防御力 + random(-3, 3))
步兵28攻: 对步兵(防8)≈20伤, 对侦察兵(防4)≈24伤
侦察兵18攻: 对步兵(防8)≈10伤, 对侦察兵(防4)≈14伤
支援10攻: 对步兵(防8)≈2伤, 对侦察兵(防4)≈6伤
```

### 3. 评分机制
```
总分 = 敌HQ伤害×4 + 我方HQ HP×2 + 据点×120 + 存活兵力×2 + 剩余补给×1
本局估算:
  player_a(我): (0×4)+(180×2)+(4×120)+存活兵力约300×2+146 = 0+360+480+600+146 ≈ 1586
  player_b(敌): (0×4)+(180×2)+(1×120)+存活兵力约200×2+65 = 0+360+120+400+65 ≈ 945
  → 优势约641分
```

### 4. 部署规则
```
部署来源: 己方HQ或已占领的控制点
部署条件: 部署点相邻的空地(plain)，且目标不在任何单位/路障/水域上
部署单位: 同一回合部署的单元已激活，不能再行动
```

### 5. 地形关键
```
路障(不可通行): (-2,0), (2,0), (-5,-2), (5,2)
水域(不可通行): (0,±2), (-1,±2), (1,±2)
关键路径:
  - 东到西被(±2,0)路障阻断，必须绕北(-2,1)或南(-2,-1)
  - 南北(0,±4)到(0,0)被(0,±2)水域阻断，须从q≤-2或q≥2绕过
  - 中央(0,0)到南部(0,4)推荐路径: (0,0)→(1,1)→(1,3)→(0,4)
```

## 总结

### 胜利关键因素
1. **⚡ 闪电四据点（第1、2、3、4回合）** — 奠定经济碾压基础
2. **🎯 步兵群推进** — 在安全距离外高效削血击杀
3. **🏥 支援治疗中盘入场** — 保障前线单位战场存活
4. **🏰 敌方HQ从未受损** — 说明我的防线稳固，敌方从未接近
5. **💪 兵力持续优势** — 终局17单位 vs 敌方少量残兵

### 核心战术原则

> **"先占点、再推进、杀侦察、灭步兵、稳推进"**

- 前4回合集中资源占领控制点，经济优势决定后续胜负
- 步兵是性价比最高的推进单位，优先部署步兵群
- 击杀优先级：支援 > 侦察兵 > 步兵 > 重坦
- 推进时保持梯队，避免单位互相阻挡浪费行动点
- 已占控制点旁的空地是理想的前线部署点

### 一句话总结

**第24局以4据点经济和步兵群推进体系稳固取胜，验证了在Hex V2中"经济先于一切、步兵优于侦察兵"的战术理念，但未趁势占领敌方cp_b是本局最大的战术遗憾。**
