## 核心胜利策略

### 1. 快速多点抢占，建立经济优势

**关键决策:** 第2回合就抢占cp_ne(补给站+12)和cp_se(前线基地部署折扣)，第3回合占领cp_nc(维修站+8)，实现3据点经济(30/回合)。

```
Turn 2: scout(7,-1)→(6,-4) 占领cp_ne
        scout(6,1)→(2,4) 占领cp_se
Turn 3: scout(5,-3)→(2,-4) 占领cp_nc → 3据点在手
```

**为什么有效:**
- 第3回合开始收入30/回合 vs 敌方12/回合(cp_nw单据点)
- 15回合经济差：(30-12)×13 ≈ 234补给
- 经济优势转化为更多部署和战场持续力

### 2. 南北双路夹击，牵制敌方兵力

**关键决策:** 北线4个侦察兵沿r=-3/-4线向西推进，南线ranger/rangers在中部活动迫使敌方分兵防守。

```
北线: scouts (6,-4)(5,-3)(4,-3)(3,-4)→逐步西推→(-7,-1)(-6,-1) 到达敌方HQ旁
南线/中线: ranger + infantry 在(7,-1)(6,1)区域保护HQ
```

**为什么有效:**
- 敌方被迫在中央部署2个单位拦截北线，削弱了南线防御
- 北线侦察兵绕过中央blocker/水域地形，从r=-4/-3通道突破
- 敌方无法同时防守两条进攻路线

### 3. 全力保ranger，发挥射程优势

**关键决策:** 第6回合ranger残血(22hp)时立即撤退，第7/12/13回合多次用support治疗ranger，使其存活到终局。

```
Turn 6: ranger(5,-1)撤至安全区
Turn 9: deploy support(8,-1)
Turn 12: heal ranger 35→61
Turn 13: heal ranger 61→72
```

**为什么有效:**
- ranger 44攻击力 + 3格射程 = 极强的远程输出
- 残血ranger仍然可以在后方安全输出
- 存活ranger大幅增加我方军队存活价值（armyValue=508 vs 敌方206）

### 4. 多回合持续施压HQ，积累伤害

**关键决策:** 从第11回合开始，每回合2个侦察兵攻击敌方HQ，每回合稳定造成15-20点伤害。


## 失误与改进

### 失误1: 北方推进路线被地图边界阻挡，浪费数次行动

**问题:** 第4-6回合，试图沿r=-4线直达(-7,-4)等位置，但地图半径限制导致部分格子超出边界或路径被水域阻断。多次尝试(-6,-4)因半径8限制失败。

**根因:** 对dual-lanes地图边界r=-4/-3和中心水域(0,-1)(1,-1)(2,-1)等地形理解不足，北方路线并非直线可达。

**改进:** 
- 开局前先确定敌方HQ坐标为(-7,0)
- 推进路线选择r=-3/-2格(有足够宽度)，而非紧贴边界r=-4
- 第10回合就应派scout探清路线而非第5-9回合反复尝试

**预期效果:** 可提前2-3回合到达HQ攻击位，HQ伤害可从66增至100+

### 失误2: HQ防守薄弱，差点被敌方翻盘

**问题:** 第7回合前，敌方scout(5,-3→6,0)直接攻击HQ，造成12伤害(93→81)。第11-14回合敌方步兵+scout多次攻击HQ，HQ最低降至19HP！

**根因:** 早期重进攻轻防守，未在HQ前预留足够防御力量。

**改进:**
- 部署时保留1-2个单位专职HQ防守
- 在HQ相邻格(6,1)、(7,1)预先部署步兵作为屏障

**预期效果:** HP受损减少，裁判分数中HQ HP项(38→200+)可增加160+分

### 失误3: 未能击杀敌方north flank单位

**问题:** 敌方ranger(4,-3→6,-2)和heavy(-3,-4)从北部威胁我方侦察兵路线，但我方选择绕过而非消灭。如果这些单位存活到后期攻击我方HQ路线，后果严重。

**根因:** 北线单位选择避战而非交战，部分原因是单位HP不足且缺乏支援火力。

**改进:**
- 在北方路线部署ranger+infantry组合，消灭敌方拦截单位
- 或至少牵制敌方北线单位，防止其南下包夹我方进攻HQ的scouts

### 失误4: 未能在第15回合摧毁HQ

**问题:** 最终HQ 114HP存活，被迫进入裁判。如果有更多侦察兵或其他攻击单位能同步攻击HQ，可能直接摧毁。

**根因:** 只有2个侦察兵能到达HQ攻击位。其他单位(infantry, ranger)距离太远无法支援。

**改进:**
- 开局面就规划多条攻击HQ路线
- 在北线保留3-4个scout轮流攻击HQ
- 或部署第二个ranger提供远程火力支援(射程3)

**预期效果:** 直接摧毁HQ = HQ victory，无需依赖裁判

## 游戏机制深度理解

### 1. dual-lanes地图(类型化据点)核心机制

```
cp_nw(-2,-4): supply +12收入
cp_nc(2,-4): repair +8收入 + 受伤单位修复
cp_ne(6,-4): supply +12收入
cp_sw(-6,4): forward_base +8收入 + 部署折扣8
cp_sc(-2,4): repair +8收入
cp_se(2,4): forward_base +8收入 + 部署折扣8

中心blocker(0,0)(1,0)(-1,0)(2,0)(-2,0) + 水域线分割南北
```

**关键发现:** 类型化据点地图中，收入不再统一，需要争夺高价值cp。本局cp_nw/cp_nc/cp_ne/cp_se各提供不同价值。北线(我方右侧)集中了cp_ne/cp_nc，有利于我从北线扩展。

### 2. 地图边界对北方进攻路线的影响

```
半径8 → max(|q|, |r|, |s|) ≤ 8 where s = -q-r

r=-4线上的格子: (-4,-4)(-3,-4)(-2,-4)(-1,-4)(0,-4)(1,-4)(2,-4)(3,-4)(4,-4)(5,-4)(6,-4)(7,-4)
  → 边界限制: (7,-4)的s=-7-(-4)=-3, |s|=3 OK, 但(7,-4)相邻敌方HQ需要多步
  → 最佳路线: (-7,-1)或(-6,-1)邻接HQ(-7,0)，s=7-1=6 OK
```

**关键发现:** 直接沿r=-4线无法快速到达HQ。最佳路线是在r=-2/-3高度通过(-6,-2)→(-7,-1)或(-6,-1)。

### 3. 裁判计分规则与本局得分

```
裁判公式: enemyHQdamage×4 + ownHQhp×2 + controlPoints×120 + armyValue×2 + supplies×1

本局得分:
- enemyHQdamage: 66 × 4 = 264
- ownHQhp: 19 × 2 = 38
- controlPoints: 3 × 120 = 360
- armyValue: 508 × 2 = 1016
- supplies: 206 × 1 = 206
- 总计: 264 + 38 + 360 + 1016 + 206 = 1884 (实际记录1794, 计算差异可能来自armyValue精算)

敌方得分:
- enemyHQdamage: 161 × 4 = 644
- ownHQhp: 114 × 2 = 228
- controlPoints: 1 × 120 = 120
- armyValue: 206 × 2 = 412
- supplies: 47 × 1 = 47
- 总计: 644 + 228 + 120 + 412 + 47 = 1451 (实际记录1421)
```

**关键发现:** 我方胜利关键在armyValue(1016 vs 412)和controlPoints(360 vs 120)。HQ HP项我方极低(38 vs 228)，但存活单位数量优势弥补了这一劣势。

## 总结

### 胜利关键因素
1. **经济腾飞**: 3据点(30/回合)奠定资源基础

**"开图抢点、北方突破、全线存活"**

### 一句话总结

**开局快扩抢双点、北方迂回打HQ、全线存活稳裁判 — 经济碾压+多点进攻是我在本局最核心的致胜法宝。**
