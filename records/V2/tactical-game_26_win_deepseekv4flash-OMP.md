## 核心胜利策略

### 1. 双线抉择地形解读 — 上下两路同时抢占

**关键决策:** 理解双线抉择地图中心被路障(q=-2..2,r=0)和水域(r=±1)隔离成南北两条独立战线，第一回合就双向布兵。

```
Turn 1: 步兵(-8,0)北线 + 步兵(-8,1)南线 — 双线起步
Turn 2: 侦察兵北上占cp_nw，步兵南下探路
Turn 3: 侦察兵再占cp_nc，步兵夺占cp_sw — 上下两路各得1个CP
```

**为什么有效:**
- 双线布阵迫使敌方必须防守两条战线，兵力分散
- 北线走经济路线（cp_nw补给→cp_nc维修），南线走战斗路线（cp_sw前线基地折扣）
- 南北两路各自独立推进，不会互相阻挡 — 解决了第21局"单位互相阻挡"的教训

### 2. 前线基地折扣部署 — cp_sw的杠杆效应

**关键决策:** 占领cp_sw（前线基地，部署折扣-8）后，将所有重要单位从这里部署，大幅节省补给。

```
cp_sw折扣效果:
  重型兵: 92→84（省8）  远程兵: 78→70（省8）
  步兵: 45→37（省8）    侦察兵: 38→30（省8）

本局从cp_sw部署:
  - 远程兵×2: 省16补给
  - 重型兵×3: 省24补给
  - 步兵×2: 省16补给
  - 合计节省: 56补给 ≈ 多出一个重型兵！
```

**为什么有效:**
- 双线抉择地图南线有cp_sw（前线基地）和cp_se（前线基地），天然适合折扣部署
- 节省的56补给意味着多部署1个重型兵 + 少量零头
- 前线基地就在战场附近，新部署单位比从HQ出发快2-3回合到达前线

### 3. 远程兵远程压制体系

**关键决策:** 第5回合部署远程兵(44攻/3射程)，在第6回合就投入战斗，形成安全输出点。

```
常规击杀流程:
Step 1: 远程兵从3格外狙击 — 对步兵造成~37伤(44-8±3)
Step 2: 步兵移动到相邻位 — 免费近战补刀19-25伤
Step 3: 此组合一次激活即可击杀一个满血步兵(56-62总伤)
```

**为什么有效:**
- 远程兵44攻击是游戏最高单体伤害，3格射程几乎无反击风险
- 配合步兵补刀可以"一次激活、两次伤害"（移动→攻击、攻击）
- 本局远程兵共造成: 63→30 + 100→63 + 63→25 + 72→43 + 43→15+击杀 ≈ 200+总伤害

### 4. 维修站治疗链 — 前线续航

**关键决策:** 占领cp_nc（维修站，每回合修复10HP友军）后，让支援兵在维修站附近活动，形成双重复合治疗。

```
治疗体系:
  cp_nc(北线维修站): 每回合自动修复10HP ⇒ 支援兵(2,-4)在站上回满
  支援兵手动治疗: 22HP/次(±6) ⇒ 步兵(0,-3)从49→100满血
  支援兵+维修站组合: 一回合最多恢复10(被动)+22(主动)=32HP
```

**为什么有效:**
- 双线抉择地图的维修站(kind="repair")让前线部队获得持续恢复能力
- 支援兵紧跟主力兵团，步兵从49→100满血后继续战斗了7回合
- 如果没有维修站+支援的治疗链，我的步兵在被包围时早就阵亡了

### 5. 重型兵集群 — 不可阻挡的铁壁

**关键决策:** 先后部署3辆重型兵(150HP/38攻)，形成重坦集群碾压敌方。

```
重坦集群时间线:
  Turn 8: 第一辆重坦(-7,4)部署 — 84补给
  Turn 12: 第二辆重坦(-5,4)部署 — 84补给
  Turn 14: 第三辆重坦(-6,3)部署 — 84补给
  Turn 15: 第四辆重坦(-5,3)部署 — 84补给

**为什么有效:**
- 150HP可以抗住几乎所有攻击，敌方仅剩的步兵和侦察兵完全无法威胁
- 38攻对轻甲单位(侦察兵防4)→34±3伤，接近秒杀脆皮
- 多辆重坦堆叠推进，敌方重型兵被集火后迅速崩溃

## 失误与改进

### 失误1: 支援兵暴露被击杀

**问题:** 第14回合支援兵留在(-2,-4)cp_nw上被敌方围攻击杀，造成了本局唯一永久减员。

- 支援兵从第2回合开始一直在前线活跃，但后期我没有及时把它撤回安全区域
- cp_nw位于北线暴露位置，敌方可以从(-2,-3)、(0,-4)等多角度接近

**改进:** 支援兵血量低于50%时应立即撤回据点防御圈内，或者跟随重坦集群移动获得掩护。

**收益:** 避免减员可保持治疗能力 + 保留60分兵力价值

### 失误2: 未在回合末巩固北线据点

**问题:** 尽管北线据点(cp_nw, cp_nc)在第2-3回合被占领，但后期侦察兵离开后没有派兵驻守，导致敌方反扑夺回。

- 第14回合支援兵被击杀后，cp_nw和cp_nc接连被敌方占领
- 最终据点数是4:2，如果守住北线应该是6:0

**改进:** 占领据点后应至少留一个单位在附近巡逻，5行动点中安排1次"换防"行动。

**收益:** 多2个据点 = 180评分 + 20补给/回合

### 失误3: 未能攻击敌方HQ

**问题:** 整局未对敌方HQ(7,0)造成任何伤害，错失了4分/伤的HQ伤害评分。

- 南线单位推进到(4,4)、(4,2)后没有再向敌方HQ突进
- 最后3回合应该让重型兵和远程兵向正东推进

**改进:** 决胜局前2回合应预留行动力向敌方HQ冲刺，哪怕只造成1伤害也能得到4评分。

**收益:** 如果造成30伤HQ = 120评分，最终比分差距会从1023扩大到1143

```

### 胜利关键因素
1. **🏗️ 双线速占点（4据点）** — 第2回合首占、第3回合3据点、第5回合4据点

### 核心战术原则

> **"双线布阵、前线折扣、远程狙击、重坦压阵、据点养经济"**

- 双线地图必须南北同时布兵，单线推进会让另一侧据点被敌轻松占领
- 前线基地的折扣效果是长期杠杆，每多一次部署就省8补给
- 远程兵是最安全高效的输出点，优先保护
- 重坦集群成型后几乎无敌
- 占领据点后必须留兵驻守，否则会被反夺

### 一句话总结

**双线抉择地图的第一场胜利验证了"先占点养经济、前线基地打折部署、多兵种协同"的战术体系，但据点巩固仍然是需要改进的薄弱环节。**
