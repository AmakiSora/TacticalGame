## 核心胜利策略

### 1. 开局抢点定经济基本盘

**关键决策:** 前4回合拿满东半场3个据点，此后收入锁定46/回合直到终局。

```
Turn 1: 斥候(6,0)→(4,3)占cp_se(补给站+20) — 收入10→30
Turn 2: 斥候(4,3)→(3,0)占cp_e(维修站+8) — 收入30→38
Turn 3: 敌斥候268a偷袭占走cp_ne
Turn 4: 击杀268a + 步兵(6,-2)→(4,-3)夺回cp_ne(前线基地+8) — 收入38→46
```

**为什么有效:**
- 全场收入642补给，是终局能连续部署13个单位（支出685）的根基
- cp_ne被偷后**下一回合立即夺回**——据点丢1回合只亏8补给，丢10回合亏80+90裁定分
- T10起步兵2ba0常驻cp_ne(4,-3)，此后我方据点再未易手

### 2. cp_e维修站杀戮区

**关键决策:** 以cp_e(3,0)为锚点，游侠炮台(2,0)+步兵墙(2,1)/(3,1)/(2,2)+支援(3,1)层层布防，让敌人在维修站射程内跟我换血。

```
维修站机制: 每回合开始为距cp 1格内【所有】受伤友军各+10hp
全场cp_e共治疗+149，敌方维修站cp_w治疗+0
敌方越墙进攻的10个单位全部阵亡，无一生还
```

**为什么有效:**
- 我的前排每回合白回10-20hp（站点+支援+奶），等效每回合白赚半次攻击
- 游侠(2,0)射程3隔着兵墙输出，敌重装（移动3）永远差一步够不到
- 敌人的支援50978f9c全场奶了98hp全给步兵6ed9399d，但没有站点叠加，效率被我完爆（我方总恢复191 vs 敌方98）

### 3. 借洞反打，一墙不拆

**关键决策:** 全场我方0次拆墙；对方重装拆墙6处((0,-3),(-1,1),(0,2),(-1,0),(0,-1),(1,-1))，我把每个洞当成漏斗口守株待兔。

**为什么有效:**
- 拆墙每段花1AP还要重装贴墙站位，对方6次拆墙≈白送我1个多回合的行动量
- 洞口一次只能过1个单位，敌人被迫添油战术，正中杀戮区下怀
- 最后对方拆出的(0,-3)洞反而成了我奔袭斥候直插敌后的高速公路

### 4. 集火算术：min伤害之和≥目标HP才动手

**关键决策:** 每次围杀前先算伤害下限（攻击-防御-3），确保当回合击杀、不给对方治疗机会。

```
Turn 9: f56c(min19)斩25934(19hp) + b97c(min28)斩263b(26hp) — 双杀
Turn 13: 目标4be余78hp，四连击25+14+16+29=84 — 新部署游侠92cc落地补刀
（部署单位当回合不能移动但【可以攻击】——这个机制救了T13）
```

**为什么有效:**
- 敌方支援每回合能奶+23~28，残血不补刀=白打一轮
- 全场击杀10个换4个，我方44次攻击均伤26.6，敌方38次均伤23.8——交换效率的核心不是均伤而是**不浪费任何一击**

### 5. 奔袭斥候双偷 + 终局补给倾销

**关键决策:** T12花30部署斥候387886e1(4,-4)，T13直插5格到(-1,-3)，T14占cp_nw、T15残血4hp再偷cp_w；同时终局把补给全部换成部署单位。

```
奔袭路线: (4,-4) →T13→ (-1,-3)【穿(0,-3)墙洞】 →T14→ (-4,-3)占cp_nw →T15→ (-3,0)占cp_w
敌方应对: T14部署守卫斥候95c4在(-3,-3)、T15部署81943在(-3,-1) — 都站在据点【旁边】而不是【上面】，拦不住
账本: 30补给的斥候 → 据点差从3:3变5:1 = +360裁定分，还迫使对方浪费76补给部署两个守卫
终局倾销: T14-T15共花112部署3个新兵 — 补给×1分，军力×2分且按【基础造价】计，折扣部署稳赚
```

**为什么有效:**
- **据点所有权持久化**：占领后离开不掉——斥候5移动力可以连环偷点
- 我是后手（Player B），第15回合我行动完直接裁定，对方对最后一偷**零反制机会**
- 裁定分差813中，据点差贡献360、军力差贡献450——这两项全部来自上述策略

## 失误与改进

### 失误1: 开局双重装在T7/T8接连阵亡

**问题:** 重装9f88、4a44先后单独暴露在敌游侠(射程3)+重装的联合火力圈里，两回合内损失184补给的主力。
**改进:** 重装换血前先数"敌方下回合最大集火输出"（游侠44+重装40+3浮动≈90），血量低于该值就必须后撤到墙后或兵墙之后。
**收益:** 若两台重装存活到中期，T12-T13对4be的围杀可提前2回合，军力分再多约140。

### 失误2: 游侠b97c3109站位差1格，被40点整斩杀

**问题:** T12我让40hp的游侠从(2,0)前移参与围攻，回合结束停在敌重装4be（移动3+近战）的打击范围内，T13被一击斩杀（40伤打40hp，一点不剩）。78补给白丢。
**改进:** 脆皮单位的安全线=敌最大移动+射程+最大伤害；残血游侠宁可这回合少输出28，也不能站进重装3格圈。
**收益:** 游侠存活则终局军力+78×2=156分，且T13不需要紧急花70补部署92cc。

### 失误3: 基于过时棋盘规划，攻击指令报"attacker not found"

**问题:** T13我按上一回合的记忆下达b97c的攻击指令，但它已在敌回合阵亡——我没有核对最新存活列表就开始规划。
**改进:** 每回合行动前必须重新拉取当前state，逐一确认计划中每个attacker/target的存活与位置，再计算集火方案。
**收益:** 本局靠临时改用步兵+新游侠补救成功；换成更紧的局面，浪费的AP可能就是胜负手。

### 失误4: 第15回合最后1AP治疗操作失败

**问题:** 用了凭记忆猜的支援旧UUID，move报"unit not found"，重试后heal又"target out of range"，白白浪费终局1AP。
**改进:** 同失误3——ID必须从当前state现取；另外按军力值公式，治疗的裁定分ROI=造价/满血×2/hp：游侠1.08分/hp>重装0.61>斥候0.58>步兵0.45——终局奶4hp的奔袭斥候或残血重装收益最高。
**收益:** 约10-20裁定分。小账，但评分决胜的局要锱铢必较。

## 游戏机制深度理解

### 1. 经济与裁定模型
```
基础收入: 10补给/回合
据点收入: 补给站+20(cp_se/cp_nw)，维修站+8(cp_e/cp_w)，前线基地+8(cp_ne/cp_sw)
裁定分 = 敌HQ伤害×5 + 己方HQ余血×5 + 据点数×90 + 军力值×2 + 补给×1
本局验证: 我方 0+500+450+992+7=1949；敌方 0+500+90+542+4=1136
```

### 2. 军力值公式（本局逆推破解）
```
军力值 = Σ(单位基础造价 × 当前HP / 满HP)
验证: 我方12个存活单位加权和=496.9→496 ✓，敌方6个满血单位=271 ✓ 完全吻合
推论①: 治疗直接挣裁定分（游侠每hp值1.08分，最高）
推论②: 折扣部署按基础造价计分——前线基地1格内部署省8补给还按原价算军力，稳赚
推论③: 终局把补给(×1)换成满血单位(×2)近乎双倍兑换，最后两回合我倾销112补给
```

### 3. 攻击与部署机制
```
伤害 = max(1, 攻击力 − 防御力 + random(-3,3))
部署: 任意己方据点1格内均可部署（不限HQ）；前线基地范围内-8折扣
部署单位当回合【不可移动、可以攻击】— T13落地补刀的依据
```

### 4. 据点与维修站机制
```
占领: 回合结束时单位站在点上即易主；【离开后所有权保持】— 奔袭连偷的根基
维修站: 每回合开始治疗己方1格内【所有】受伤单位各+10（cp_e全场为我+149）
```

### 5. 墙体机制（breach地图核心）
```
中央石墙纵贯战场，仅重装可拆（1AP/段）
本局对方拆6处: (0,-3),(-1,1),(0,2)由双重装T2-T6拆出，(-1,0),(0,-1),(1,-1)由4be在T9-T11拆出
我方拆0处——防守方让进攻方付开门钱，洞口就是天然漏斗杀阵
```
