## 核心胜利策略

### 1. 闪电占点 — 5回合控制全部据点

**关键决策:** 第1-5回合以占领控制点为最高优先级，利用侦察兵的高机动性快速抢点。

```
Turn 1: 侦察兵(-7,-1)→(-4,0) 占领cp_a
Turn 2: 侦察兵(-4,0)→(0,-4) 占领cp_n  
Turn 3: 步兵(-1,1)→(0,0) 占领cp_c
Turn 4: 侦察兵(1,-1)→(4,0) 占领cp_b
Turn 5: 侦察兵(4,0)→(0,4) 占领cp_s
```

**为什么有效:**
- 每个据点+15补给/回合，5个全占后每回合收入85补给（基础10+75）
- 敌方从第5回合起只有10-25补给/回合，经济被彻底碾压
- 高收入让我可以在后期累计805补给，而对方只有10

### 2. 侦察兵闪电战 — 利用高机动性

**关键决策:** 使用侦察兵（移速5）快速穿插占领远处据点，步兵（移速3）占领近处据点。

```
侦察兵路径:
  (-7,-1)→(-4,0)→(0,-4)→(1,-1)→(4,0)→(0,4)→(5,3)→(7,1)
  — 横跨整个战场，从左侧到右侧攻击HQ！
```

**为什么有效:**
- 侦察兵移速5，是步兵移速3的1.67倍
- 可以一回合从地图一侧穿越到另一侧
- 虽然攻击力低（18攻/4防），但占领能力与步兵相同
- 非常适合快速占领据点、骚扰后方

### 3. 游骑兵远程压制 — 核心输出单位

**关键决策:** 游骑兵（42攻/射程3）作为主要输出来源，从安全距离攻击敌方单位和HQ。

```
游骑兵部署位置:
  (-8,1)→(-6,1)→(-4,1)→(-2,1)→(0,1)→(1,0)→(3,0)→(5,0)→(6,0)
  — 沿中线稳步推进，每回合都保持攻击距离
```

**为什么有效:**
- 游骑兵42攻是游戏中最高（步兵28、重型36）
- 射程3可以站在步兵/重型后面安全输出
- 对HQ伤害可观（28-42/次），是攻HQ的主力
- 最终由游骑兵在(6,0)完成最后一击

### 4. 集中火力逐个击杀

**关键决策:** 每回合集中2-3个单位攻击同一个敌人，确保击杀而不是分散伤害。

```
Turn 3: 游骑兵+步兵集火敌方侦察兵 → 46HP残血
Turn 4: 步兵(0,1)+游骑兵(-2,1)+侦察兵(1,-1) 集火击杀敌方侦察兵
Turn 5: 继续攻击敌方受伤单位
```

**为什么有效:**
- 每个单位单次攻击约20-35伤害，单个难以击杀100HP步兵
- 集中2-3个单位可确保一回合击杀
- 减少敌人反击机会，降低己方伤亡

### 5. 持续压制 — 不给对手喘息机会

**关键决策:** 占领全部据点后立即向敌方HQ推进，不给对手任何反击窗口。

```
Turn 7: 首次攻击HQ（侦察兵+游骑兵，50伤害）
Turn 8-15: 每回合持续攻击HQ
Turn 16: 游骑兵最后一击，HQ摧毁
```

**为什么有效:**
- 从第7回合到第16回合，共10个回合持续攻击HQ
- 每回合攻击2-3次，累计造成240+伤害
- 不让敌方有重建经济和组织反击的时间

## 失误与改进

### 失误1: 第1回合部署游骑兵花费过高

**问题:** 第1回合部署游骑兵花75补给，导致剩余仅5，无法部署其他单位。
**改进:** 第1回合部署步兵（45补给）性价比更高，剩余35可用于第2回合部署侦察兵继续抢点。
**收益:** 保留更多补给应对后续部署，步兵在前线扛线效果更好。

### 失误2: 侦察兵突进过深导致残血

**问题:** 第7回合侦察兵(7,1)和(7,0)攻击HQ后暴露在敌方阵地，后续被反击打残（最终只剩7HP）。
**改进:** 侦察兵防御低（4防），攻击HQ后应尽量撤退到安全位置，或由步兵/重型完成HQ攻击。
**收益:** 保留侦察兵的机动性和占领能力，继续执行其他任务。

### 失误3: 未部署支援单位

**问题:** 整局游戏没有部署任何支援单位（治疗），导致部分单位残血后无法恢复。
**改进:** 当有3个以上单位受伤时，应部署支援单位提供治疗（28奶量），维持部队续航。
**收益:** 减少单位阵亡，延长前线作战时间。

## 游戏机制深度理解

### 1. 经济模型（Hex V2）
```
基础收入: 10补给/回合
控制点收入: 15补给/控制点/回合
最大收入: 10 + 15×5 = 85补给/回合
初始补给: 80
```

### 2. 攻击机制
```
伤害 ≈ 攻击力 - 防御力 + random(-3, 3)，最低1伤害
近战单位被攻击后可以反击
HQ不会反击
```

### 3. 控制点占领机制
```
控制点在回合结束时被站在上面的单位自动占领
仅步兵和侦察兵可以占领控制点
占领后每回合提供15补给收入
```

### 4. 行动点机制
```
每回合最多5个行动点
部署/移动/攻击/治疗均消耗行动点
已激活的单位可以继续免费行动
部署即激活，新部署单位不能移动
```

## 总结

### 胜利关键因素
1. **闪电占点** — 第1回合就占领据点，第5回合控制全部5个，建立经济碾压
2. **侦察兵机动** — 利用移速5快速穿越战场，同时承担占领和骚扰双重角色
3. **游骑兵远程** — 42攻/射程3是对HQ和单位的核心输出，部署后可持续作战
4. **持续压制** — 第7回合即开始攻击HQ，不给对手任何重建经济的时间
5. **集中火力** — 每回合集火击杀一个敌人，减少己方伤亡

### 核心战术原则

> **"快占点、远输出、不停压"**

- **前5回合**：所有行动以占领控制点为最高优先级
- **中盘**：游骑兵远程推进，侦察兵侧翼穿插
- **后期**：集中火力围攻HQ，不给对手喘息机会

### 与第2局对比的改进

1. 第2局在Turn 13才大量暴兵终结，本局更早（Turn 7）开始攻击HQ
2. 第2局部署了10个游骑兵，本局部署了5个，更加精打细算
3. 两局都实现了无伤胜利，策略成熟可靠

### 一句话总结

**通过第5回合前占领全部5个控制点建立经济碾压，利用游骑兵远程火力从第7回合起持续攻击HQ，第16回合无伤摧毁敌方HQ取得完胜。**
