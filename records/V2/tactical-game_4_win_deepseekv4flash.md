## 核心胜利策略

### 1. 双侦察兵开局 — 快速占点

**关键决策:** 第1回合部署2个侦察兵，利用侦察兵移速5快速抢占地图。

```
Turn 1: 步兵(7,0)→(4,0)占领cp_b
        部署侦察兵(7,0)、(8,-1)
Turn 2: 部署侦察兵(3,0)、(4,-1)
        侦察兵(5,0)→(1,0)
```

**为什么有效:**
- 侦察兵移速5，部署后可以快速到达战场任何位置
- 开局4个侦察兵可以同时执行占点、侦察、骚扰多个任务
- 侦察兵成本仅40，性价比极高

### 2. 步兵+侦察兵混合推进 — 多点开花

**关键决策:** 步兵走中线稳步推进，侦察兵从上下两翼包抄，形成三路压迫。

```
步兵路线（中线）:
  (7,0)→(4,0)→(2,-1)→(0,-1)→(0,0)→(-2,-2)→(-4,0)→(-5,0)

侦察兵路线（上翼）:
  (7,1)→(5,0)→(1,0)→(0,1)→(-1,1)→(-2,1)→(-4,1)→(-5,1)

侦察兵路线（下翼）:
  (7,-1)→(5,-1)→(3,-1)→(1,-1)→(0,-4)→(-3,-1)→(-4,-1)→(-5,-1)
```

**为什么有效:**
- 三路推进让敌方难以防守所有方向
- 侦察兵利用移速优势从侧翼包抄，步兵在中线扛线
- 最终5个游骑兵从不同角度围攻HQ，让敌方无法全部拦截

### 3. 游骑兵海 — 远程火力压制

**关键决策:** 从中盘开始大量部署游骑兵（42攻/射程3），作为核心输出。

```
游骑兵部署:
  Turn 8: 游骑兵(-1,-4)
  Turn 11: 游骑兵(-1,-4)、游骑兵(-1,-3)
  Turn 14: 游骑兵(-1,-4)
  Turn 17: 游骑兵(-5,0)
  
最终5个游骑兵同时在场，从(-5,0)、(-5,-1)、(-4,-2)、(-6,-2)围攻HQ
```

**为什么有效:**
- 游骑兵42攻是游戏中最高，射程3可以安全输出
- 多个游骑兵可以同时攻击同一目标，快速摧毁
- 最终回合3个游骑兵同时攻击HQ，一回合打出致命伤害

### 4. 集中火力逐个击杀

**关键决策:** 每回合集中2-4个单位攻击一个敌人，确保击杀。

```
Turn 3-4: 步兵×2+侦察兵×2 集火敌方步兵 → 击杀
Turn 6-7: 步兵+侦察兵×3 集火敌方单位 → 击杀
Turn 8-9: 步兵+侦察兵×4 集火敌方单位 → 击杀
Turn 11-13: 游骑兵×3+侦察兵 集火敌方单位 → 击杀
```

**为什么有效:**
- 敌方单位HP 70-100，单个单位伤害20-35，需要2-4个单位才能一回合击杀
- 集中火力减少敌人反击机会
- 击杀后减少敌方输出，保护己方单位

### 5. 侦察兵转职"肉侦"

**关键决策:** 后期侦察兵不参与攻击，而是利用高移速站到控制点上提供视野和经济收入。

```
Turn 17-18: 侦察兵(0,4)站在cp_s上
            提供南部视野和15补给/回合收入
```

**为什么有效:**
- 侦察兵防御低（4防），攻击HQ容易被反击打残
- 不如让游骑兵输出，侦察兵占点提供经济和视野
- 保留侦察兵用于后续扩张或应对突发情况

## 失误与改进

### 失误1: 侦察兵过度参与攻击导致残血

**问题:** 多个侦察兵在前中期参与近战攻击，被反击打残。最终侦察兵(0,4)仅剩25/70HP。
**改进:** 侦察兵应专注于占领和机动，攻击任务交给步兵和游骑兵。侦察兵18攻/4防，攻击效率低且容易受伤。
**收益:** 保留侦察兵满血用于快速占点和侧翼穿插。

### 失误2: 步兵突进过深被孤立

**问题:** 步兵(-2,-2)在Turn 6-7突进到(-4,-1)附近，周围没有友军支援，被敌方集火。
**改进:** 步兵推进时应保持阵型，确保相邻格有友军可以支援。步兵移速3，游骑兵移速2，需要协调推进速度。
**收益:** 减少步兵阵亡，保持前线扛线能力。

### 失误3: 部署游骑兵位置过于集中

**问题:** 3个游骑兵部署在(-1,-4)、(-1,-3)附近，位置过于集中，容易被敌人一波反击。
**改进:** 游骑兵应分散部署在不同位置，从多个角度输出。分散部署也可以覆盖更大的攻击范围。
**收益:** 提高游骑兵生存率，扩大火力覆盖范围。

### 失误4: 没有部署支援单位

**问题:** 整局游戏没有部署任何支援单位（治疗），导致受伤单位无法恢复。
**改进:** 当有3个以上单位HP低于70%时，应部署支援单位提供治疗（28奶量）。
**收益:** 延长前线单位作战时间，减少重新部署的花费。
