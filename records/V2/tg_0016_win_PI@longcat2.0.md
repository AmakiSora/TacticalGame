## 失误与改进

### 失误1: 中心区域单位过密，互相卡死

**问题:** 第8-13回合，中心区域(-2≤q≤1, -1≤r≤1)聚集了我方6-7个单位，导致大部分单位无法移动。多次出现"already moved"和"target is not reachable"错误，浪费了大量行动点。

**根因:** 
- 早期推进时，所有单位都向中心区域集中
- 没有预留机动空间
- 单位密度超过地图承载能力

**改进:** 
- 推进时保持单位间距至少2格
- 避免在3×3区域内放置超过3个单位
- 预留一条清晰的进攻通道

**预期效果:** 可节省3-4个行动点/回合，用于更有价值的操作

### 失误2: 未能充分利用已激活单位的免费行动

**问题:** 第17回合，infantry(5,-1)移动后，本可以免费继续攻击或移动，但误以为"already moved"。实际上已激活单位可以继续行动。

**根因:** 对"已激活单位免费行动"规则理解不够深入，误以为每单位只能行动一次。

**改进:** 
- 每个行动点激活一个单位后，立即规划该单位的完整行动链
- 例如：移动→攻击→再攻击，或移动→移动→攻击
- 最大化每个行动点的价值

**预期效果:** 每个行动点可以产生2-3倍的效果

### 失误3: 未能在第20回合摧毁敌方HQ

**问题:** 第20回合只有infantry(8,-1)能攻击HQ，造成23伤害（200→177），未能摧毁。如果有更多单位能同时攻击HQ，可能直接获胜。

**根因:** 
- 中心区域拥堵导致单位无法及时到达HQ附近
- 没有提前规划多路进攻HQ的路线
- 最后一回合才发动攻击，时间不够

**改进:** 
- 从Turn 15开始就规划HQ攻击路线
- 至少保持2个单位能在HQ相邻格
- 提前清理HQ周围的敌方单位
- 在Turn 18-19就发动HQ攻击，留出足够的攻击回合

**预期效果:** 可能在第18-19回合就摧毁HQ，直接获胜

### 失误4: 南侧路径误判

**问题:** 多次尝试从南侧（r=-2）绕到敌方HQ，但(0,-2)、(1,-2)都是water，无法通行。浪费了多次移动尝试。

**根因:** 没有提前检查地形，盲目相信"南侧畅通"的判断。

**改进:** 
- 推进前先检查目标格子的terrain
- 记住关键water位置：(-1,-2)、(0,-2)、(1,-2)、(-1,2)、(0,2)、(1,2)
- 提前规划可行路径

**预期效果:** 避免无效移动，节省行动点

## 游戏机制深度理解

### 1. 行动点管理（核心机制）

```
总行动点: 5/回合
激活成本: 1 AP = 激活1个单位
免费行动: 已激活的单位可以继续移动/攻击/治疗
Deploy: 总是消耗1 AP（新单位被激活，但本回合不能移动）
```

**关键发现:** 行动点是最重要的资源。每个AP都应该激活一个能产生最大价值的单位。已激活单位的免费行动是"隐藏价值"，必须充分利用。

### 2. 地形与移动

```
water格子: (-1,-2)、(0,-2)、(1,-2)、(-1,2)、(0,2)、(1,2)
blocker: (-5,-2)、(-2,0)、(2,0)、(5,2)
```

**关键发现:** 地图上有两条water带（r=-2和r=2的中间区域），将地图分割为三个走廊。r=-1和r=1是最主要的进攻通道。

### 3. 经济模型

```
基础收入: 10/回合
控制点收入: 15/回合/个
双CP总收入: 10 + 15×2 = 40/回合
```

**关键发现:** 控制点是经济核心。双CP比单CP多收入15/回合，20回合就是300补给差距。这局我从第3回合就保持双CP，经济优势明显。

### 4. 战斗机制

```
伤害 = max(1, 攻击力 - 防御力 + random(-3, +3))
ranger远程: 攻击范围3，可以隔格攻击
heal: 恢复20 HP，范围1
```

**关键发现:** ranger的3格射程是极强的战术优势。利用射程在安全距离消耗敌方，是无损作战的最佳选择。

### 5. 裁判计分规则

```
敌方HQ伤害 × 4
己方HQ HP × 2
控制点 × 120
存活单位价值 × 2
supplies × 1
```

**关键发现:** 控制点价值极高（120分），相当于1.5个heavy的价值。HQ伤害权重4倍也很重要。存活单位数量和价值是基础分数。

## 总结

### 胜利关键因素
1. **经济优势**: 第3回合即实现双CP收入，全程保持经济领先
2. **ranger远程消耗**: 利用3格射程无风险输出，累计造成200+伤害
3. **兵力优势**: 12个单位 vs 敌方4个，数量碾压
4. **HQ攻击**: 成功将infantry推进到HQ旁并造成伤害

### 核心战术原则

> **"r=1和r=-1是两条高速公路，单位间距保持2格以上，已激活单位必须榨干最后一滴行动力"**

- 进攻通道选择r=1或r=-1，避免穿越water带
- 单位间距至少2格，避免拥堵
- 每个被激活的单位都要规划完整行动链
- CP是经济核心，第3回合前必须占领2个

### 一句话总结

**在被中心区域拥堵严重拖慢节奏的情况下，凭借早期双CP经济优势+ranger远程消耗+最终infantry兵临HQ，以裁判分数2312:1140赢得首场胜利。**
