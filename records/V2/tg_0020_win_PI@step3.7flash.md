## 核心胜利策略

### 1. 东侧快速开花，建立前进基地

**关键决策:** 第1回合即把主力步兵推向东侧补给站 `cp_b`(4,0)，侦察兵侧翼掩护。

```
Turn 1: 步兵(7,0)→(4,0) — 花费0行动点，占东侧补给站
Turn 2: 侦察兵(5,1)→(5,0) — 巩固东线
Turn 3: 部署侦察兵至cp_b(4,1) — 建立前线哨站
```

**为什么有效:**
- 东侧补给站是离敌方HQ最近的控制点，占领后可直接威胁敌方右翼
- 东线plain地形开阔，便于后续部署和推进
- 我方右侧HQ(8,0)到cp_b(4,0)距离近，补给线短

### 2. 中央反复争夺，消耗敌方有生力量

**关键决策:** 第3回合用侦察兵插入中央 `cp_c`(0,0)，第7-13回合持续打击敌方中央步兵。

```
Turn 3: 侦察兵(5,0)→(1,0) — 向中央插进
Turn 7: 攻击敌方侦察兵(0,-1) — 削HP
Turn 10: 步兵(1,1)→(1,0)攻击敌方步兵 — 两路夹击
Turn 12: 步兵(1,0)再次攻击 — 持续压制
```

**为什么有效:**
- 中央 `cp_c`(0,0) 是地图中心，控制后可辐射南北
- 敌方在中央投入大量兵力来回救，消耗其行动点
- 通过连续攻击迫使敌方无法安心占点

### 3. 南线稳固后，伺机占北部中继

**关键决策:** 第5回合向南线投入ranger和infantry，稳固南部中继站 `cp_s`(0,4)。

```
Turn 5: 部署ranger至cp_b(5,0)；部署infantry至cp_b(5,-1)
Turn 6: 侦察兵(-1,0)→(0,-1) — 向北方窥视
Turn 16: 侦察兵(-2,3)→(-2,3) — 向北推进
```

**为什么有效:**
- 南部中继站一旦稳固，提供稳定 income
- 北方中继站 `cp_n`(0,-4) 是最后空白点，占满3点后 adjudication 优势明显
- 北线推进避开东线主力决战，分散敌方注意力

### 4. 重装突破，打破僵局

**关键决策:** 第8回合在东线部署heavy `(4,1)`，增强正面突破能力。

```
Turn 8: 部署heavy至cp_b(4,1) — 145HP高防御单位
```

**为什么有效:**
- heavy的145HP和14防御使其成为战线中流砥柱
- 敌方infantry攻击heavy仅造成约18-24伤害，需要多次攻击才能击杀
- heavy存在迫使敌方分散兵力应对

### 5. 资源优先用于部署，而非囤积

**关键决策:** 金币超过100后立即部署新单位，而不是攒到200以上。

```
Turn 3: 125金部署侦察兵
Turn 5: 165金部署ranger
Turn 8: 130金部署heavy
Turn 9: 110金部署infantry
```

**为什么有效:**
- 每回合income 10-25金，囤积无意义
- 早部署早形成战力，多一个单位多一个行动点
- 单位价值（45-90金）远低于控制点价值（120分）

## 失误与改进

### 失误1: 中央侦察兵孤军深入被围杀

**问题:** 第8回合中央侦察兵 `37104e6c` 在 `(0,0)` 被敌方多单位夹击，HP从34降至0阵亡。

**改进:** 侦察兵占点后应迅速回撤至己方步兵保护范围，或部署支援单位 adjacent 保护。

**收益:** 避免无意义损失，保持中央控制连续性。

### 失误2: 部署点选择多次失败

**问题:** 第5回合部署infantry至 `cp_b(5,-1)` 返回 `invalid_terrain`，第9回合部署至 `cp_b(3,0)` 返回 `invalid_terrain`。

**改进:** 部署前先用API查询目标格地形，确认是plain且无单位。

**收益:** 减少无效操作，每回合多0.5-1个有效行动点。

### 失误3: 回合末空转

**问题:** 第4、9、16回合因地形限制（blocker/water）无法推进，行动点浪费。

**改进:** 提前规划多路线备用方案，不要把所有单位压在唯一通道上。

**收益:** 保持每回合3-4个有效操作，加速占点节奏。

### 失误4: 中央反复易手，消耗过多行动点

**问题:** 第7-15回合连续9回合围绕中央 `cp_c`(0,0) 拉锯，实际有效击杀仅2次。

**改进:** 中央不宜强攻时，应分兵占北部中继站 `cp_n`(0,-4) 或南方中继站 `cp_s`(0,4)，用分点优势逼敌方分兵。

**收益:** 用多线压力替代单点消耗，adjudication阶段点数优势更明显。

## 游戏机制深度理解

### 1. 六角移动与地形

```
移动范围: infantry=3, scout=5, heavy=2, ranger=2, support=3
地形影响: water/blocker不可通行，plain可通行
六角距离: max(abs(dq), abs(dr), abs(ds))，其中s=-q-r
```

**关键发现:**
- 地图中 `(-2,0)` 和 `(2,0)` 是blocker，直接切断东西向直连通道
- 中央 `(0,0)` 虽可直达，但周边 `(0,-2)` 和 `(0,2)` 是water，形成天然屏障
- 占点必须单位站在点上，单位不能重叠

### 2. 攻击与防御机制

```
伤害公式: max(1, 攻击力 — 防御力 + random(-3, 3))
 infantry: atk=28, def=8 → 平均伤害20-21
 scout:    atk=18, def=4 → 平均伤害14-15
 heavy:    atk=36, def=14 → 平均伤害22-23
 ranger:   atk=46, def=3 → 平均伤害43-44
```

**关键发现:**
- 步兵打步兵约2-3回合击杀，打heavy需6-8回合
- ranger 46攻击力是清场利器，但移动范围仅2格，需要步兵掩护推进
- 攻击无距离衰减，但仅限attackRange=1（近战）或3（ranger）

### 3. 控制点与经济

```
baseIncome: 10/回合
controlPointIncome: 15/点/回合
adjudication权重: 控制点×120，HQ伤害×4，HQ HP×2， ArmyValue×2， Supplies×1
```

**关键发现:**
- 控制点 income 是 base 的2.5倍，占3个点收入=50/回合
- adjudication阶段控制点权重极高（120分），2个点=240分基础分
- 囤积supplies（585）在adjudication中仅585分，不如多占1个点（120分+income）

### 4. 行动点机制

```
actionsPerTurn: 5
部署消耗: 1点
已激活单位后续行动: 免费
```

**关键发现:**
- 部署是独立消耗，不能和其他行动共享行动点
- 攻击后如单位未移动，可免费移动+再攻击（二连击）
- 地形限制导致行动点浪费是本局最大效率损失

## 总结

### 胜利关键因素

1. **首回合占东侧补给站** — 建立战略桥头堡，迫使敌方回防
2. **中央反复争夺消耗敌方** — 虽效率不高，但成功牵制敌方主力
3. **东线持续增兵** — heavy+ranger组合形成正面压制
4. **得分阶段囤积金币** — 545金supplies在adjudication中提供额外分数
5. **HQ零受损** — 全程未让敌方摸到 HQ 200HP

### 核心战术原则

> **"首占要点，中央牵制，东线突破，分点压 adjudication"**

- 首回合必须抢占离HQ最近的控制点
- 中央不宜强攻时，用多线占点逼敌方分兵
- 东线plain地形适合heavy+ranger组合推进
- 第15回合后囤积supplies，准备adjudication得分

### 一句话总结

**东线开花、中央牵制、东线重装突破、adjudication囤分，以2:2点数+545囤分赢得判定胜利。**
