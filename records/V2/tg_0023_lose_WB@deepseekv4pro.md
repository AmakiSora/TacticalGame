## 六、战术得失

### 做得好的决策
1. **开局双线推进** — 侦察兵北上 + 步兵占 cp_a，效率高
2. **早期部署重型坦克** (T2) — 在资源充裕时快速建立坦克优势
3. **后期坚持补兵攒分** — 明知败局仍最大化评估分
4. **击杀了 2 个敌方单位** — 步兵+侦察兵各一

### 失误分析
1. **游骑兵部署太晚** (T14) — 应在 T8-T10 就部署，形成对等火力
2. **cp_n 防守不力** — 侦察兵孤军深入，无步兵协防
3. **重型坦克冲锋过早** — 在游骑兵射程内硬冲，被集火
4. **未部署第二个支援兵** — 一个支援兵无法维持坦克血量
5. **南线侦察兵闲置** — (0→-5,3) 后未有效争夺 cp_s

### 改进思路
- **T8 前部署至少 1 个游骑兵**，形成远程威慑
- **据点要有协防**，侦察兵占点后立刻派步兵支援
- **重型坦克不要脱离支援兵太远**，保持在治疗范围
- **经济劣势时龟缩攒分**，不要主动冲锋
- **早点放弃 cp_n 争夺**，集中兵力保卫 cp_a 和骚扰南线

## 八、经验总结

### 核心教训
1. **游骑兵是 V2 版本的核心兵种** — 射程3+46攻确保了制空权，谁游骑兵多谁赢
2. **经济=据点=胜利** — 丢据点→没钱→少兵→丢更多据点，恶性循环
3. **支援兵必须紧跟坦克** — 坦克没有治疗就是活靶子
4. **先手优势（player_a）不足以弥补经济劣势**
