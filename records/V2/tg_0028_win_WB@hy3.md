## 核心胜利策略

### 1. 开局抢点定经济基本盘

**关键决策:** 前3回合拿满己方半场3个据点（cp_nw补给站+20、cp_w维修站+8、cp_sw前线基地+8），收入从30/回合锁到46/回合；第7回合再夺回 cp_ne，收入升到54/回合直到终局。

```
Turn 1: scout(-6,0)→(-4,-3)占cp_nw(补给站+20) — 收入10→30
Turn 2: scout(-4,-3)→(-3,0)占cp_w(维修站+8) — 收入30→38
Turn 3: scout(-3,0)→(-4,3)占cp_sw(前线基地+8) — 收入38→46（锁定）
Turn 7: scout(3,-3)→(4,-3)夺回cp_ne(前线基地+8) — 收入46→54（终局）
```

**为什么有效:**
- 全场收入464补给，是终局能连续部署8个单位（支出331）的根基——经济碾压让对手始终追不平军力
- 据点加成不仅给钱，还提供**部署锚点**与（维修站）治疗光环，后期的(5,-3)/(4,-3)部署都受益于cp_ne
- 对手同样抢点（cp_se/cp_e/cp_ne），但被我中期夺回 cp_ne 后，收入差被拉开

### 2. 破墙开路——进攻方自己造血路

**关键决策:** 第2回合让 heavy 移动到石墙相邻格(-1,-2)后**同回合爆破 (0,-2)**，仅花1个行动点就撕开中央石墙唯一需要的缺口。此后重装经此洞一路直插敌腹地。

```
破墙组合: heavy(-3,-2)→(-1,-2) [move, 1AP] + 爆破(0,-2) [demolish, 同激活免费]
结果: (0,-2) 由 blocker 变 plain，重装(1,-2) 下回合即可穿洞
全场我方仅拆1段墙——一段足够，不用像第27局对手那样拆6段
```

**为什么有效:**
- 破障行动的核心就是这道中央墙；作为**进攻方**，创造自己的突破口是胜利前提（与第27局"防守方不拆墙"正相反）
- 只拆1段、且拆在己方半场一侧，对手来不及在洞后布防——洞口从第3回合起就成了我的高速公路而非漏斗
- 拆墙+移动同回合只花1AP（见机制2），效率极高

### 3. 尖刀重装直插总部（两线突破）

**关键决策:** 破墙后让一台 heavy 专职当"尖刀"，沿 (-1,-2)→(1,-2)→(3,-2)→(4,-2)→(6,-2)→(7,-1)→(8,-1) 一路滚雪球直插敌方HQ；另一台 heavy 在 (2,-2)/(4,-2)/(6,-2) 跟进，形成双线，对手堵哪边都漏另一边。

```
尖刀路线: (-1,-2)T2→(1,-2)T3→(3,-2)T4→(4,-2)T5→(6,-2)T6→(7,-1)T7→(8,-1)T8
结果: 第8回合站上(8,-1)（距HQ相邻），开始拆家；对手重装(7,-2)/(8,-2)只能被动贴防
```

**为什么有效:**
- 尖刀始终比对手反应快半拍——它每回合前进2~3格，对手回防单位永远差一步
- 双线突破让对手的集火分散：打尖刀则第二台 heavy 跟进，打第二台则尖刀继续拆
- 等对手在(8,-2)贴脸反扑削我尖刀到9血时，总部已只剩18血，一击即死

### 4. 激活效率：move+attack/demolish 同回合只花1AP

**关键决策:** 利用引擎"首次激活消耗1AP、同回合后续动作免费"的规则，把移动与攻击/拆墙 chaining 在同一点数内。

```
Turn 2: heavy move(-1,-2) + demolish(0,-2) = 1AP（非2AP）
Turn 5: heavy move(4,-2) + attack敌heavy(5,-2) = 1AP
Turn 8: heavy move(8,-1) + attack HQ = 1AP
全场因此每回合多出约1个有效动作，10回合等效多打3~4个动作
```

**为什么有效:**
- 同样5AP，别人只能动5次，我能"动+打"组合出6~7次有效操作
- 拆家节奏因此提前：若每回合老实1移动1攻击各算1AP，拆墙会晚1回合、拆家晚2回合

### 5. 终局补刀算术——确保尖刀存活到能拆

**关键决策:** 第9回合对手用 heavy(8,-2) 反击把我的尖刀(8,-1)从39削到9血，但未致死；我算清"1台敌重装1回合只能打1次（约27伤），打不死9血单位"，于是第10回合照常补刀拆家。

```
T9敌回合: 敌heavy(8,-2)攻击我heavy(8,-1) 39→9（1击，未致死）
T10我方: heavy(8,-1)攻击HQ 18→0 — 攻击不依赖自身HP，9血照样拆光
```

**为什么有效:**
- 拆家只要求"尖刀站在(8,-1)且存活"，不要求满血——对手1回合的反扑不足以击杀，胜利已锁定
- 第9回合我同步让 infantry(5,-3)→(4,-3) 重占 cp_ne，确保即便拖入评分也不会丢据点

## 失误与改进

### 失误1: 第7回合 axial 距离算错，浪费一次拆家机会

**问题:** 我误以为 (7,-1) 与 HQ(8,0) 相邻，让重装移动到 (7,-1) 后发动攻击，引擎报 `out of range (2 > 1)`——实际 hex axial 距离 = (|dq|+|dr|+|dq+dr|)/2 = (1+1+2)/2 = **2**，并不相邻。重装已 moved 无法再动，本回合拆家落空，总部多撑了1回合。
**改进:** 每次下达 move/attack 前，先用公式 `dist = (|q1-q2| + |r1-r2| + |(q1+r1)-(q2+r2)|)/2` 校验距离，确认 attack 目标距离 ≤ 单位攻击范围（heavy=1）。正确的拆家位是 (8,-1)（与HQ距离1）。
**收益:** 若第7回合就站上(8,-1)，总部第8回合即降到22血、第9回合已拆光，可提前1~2回合结束。

### 失误2: 部署/攻击坐标规划粗糙，多次无效动作

**问题:** 第5回合第5个动作部署 scout@(-7,1)，因距 HQ(-8,0) axial 距离=2（非相邻）被拒（out_of_deploy_range），浪费1AP；第6回合 scout(-7,0)→(-2,0) 因路径被己方 infantry(-3,0) 阻挡报 `target not reachable`；同回合 heavy(6,-2) 攻击 (5,-3) 因距离2失败。
**改进:** 部署前确认目标格与 HQ/已占据点 **距离=1**；移动前确认路径不被己方单位挡死（或绕行）；攻击前确认目标在射程内。本局靠即时诊断改格补救，但浪费了若干AP。
**收益:** 每省1个无效AP，等于多一次有效推进/输出，对紧凑的进攻节奏影响明显。

### 失误3: 斥候过量部署，部分成炮灰

**问题:** 为抢点连续部署3个 scout（T3/T5/T6），其中一个在 cp_ne 被对手反击打死（T9 时 hp0），投入的38补给未转化为持久价值。
**改进:** 抢点用 scout 没问题，但夺点后应换更肉的 infantry 常驻（如第9回合做的 infantry(5,-3)→(4,-3)），而非让脆皮 scout 长期站点。点占下后要"换防"。
**收益:** 减少无谓单位损耗，把补给更多地倾斜给能终结比赛的重装。

## 游戏机制深度理解

### 1. 经济模型
```
基础收入: 10补给/回合
据点收入: 补给站+20(cp_nw/cp_se)，维修站+8(cp_w/cp_e)，前线基地+8(cp_sw/cp_ne)
本局验证: 1点→30, 2点→38, 3点→46, 4点→54，与读数完全吻合
```

### 2. 行动点 / 单位激活（本局核心效率来源）
```
规则: 单位首次激活(任意 move/attack/demolish)消耗1AP；
      同回合内若已 move（只设 hasMoved，不设 hasActed），再 attack/demolish 免费
      → move+attack = 1AP，move+demolish = 1AP
验证: 第2回合 move(-1,-2)+demolish(0,-2) 仅 actionsUsed+1；第8回合 move(8,-1)+attackHQ 同理
推论: 每回合5AP可打出"5次激活"，其中若干个可附带1次免费攻击/拆墙，等效6~7次操作
```

### 3. 墙体机制（breach 地图核心）
```
中央石墙: q=0 整列 blocker，q=±1 部分格也是 blocker，仅重装可拆（每格1AP）
拆墙条件: 重装与目标墙格距离≤1，且 move 到相邻格后同回合 demolish
本局只拆 (0,-2) 1段——一段足够打通己方半场到敌半场的走廊
对比第27局: 对手拆6段当漏斗口；本局作为进攻方，自己开1段洞直插，范式相反
```

### 4. 攻击 / 部署机制
```
伤害 = max(1, 攻击力 − 防御力 + random(-3,3))
heavy 攻击力40、防御13、移动3、攻击范围1、造价92
HQ: 100HP，0防御 → heavy 每击约37~43伤，3击摧毁
部署: 任意己方据点/HP 1格内可部署；前线基地范围内-8折扣
      部署单位当回合【不可移动、可以攻击】
```

### 5. 据点 / 维修站机制
```
占领: 回合结束时单位站在点上即易主；离开后所有权保持（scout 可抢完即走）
维修站: 每回合开始治疗己方1格内受伤单位各+10（本局我方 cp_w 提供治疗，但尖刀在敌半场用不上）
```

## 总结

### 胜利关键因素
1. **经济先行**: 前3回合拿满3据点锁46/回合收入，第7回合夺回 cp_ne 升到54，总收入464支撑8次部署

3. **尖刀直插**: 一台 heavy 专职滚雪球到(8,-1)，双线突破让对手防不胜防，第8~10回合3击拆光HQ

### 核心战术原则

> **"抢点养经济，破墙开血路，尖刀直插拆总部"**

- 据点是本体：收入、部署锚点、治疗全在点上，前3回合必须拿满己方半场

**作为进攻方，我第2回合撕开中央石墙唯一需要的那个洞，让一台重装顺着它一路滚到敌方HQ脸前，第8~10回合三记重拳把100血总部砸成0——破障行动的胜负，从破墙那一刻就写好了。**
