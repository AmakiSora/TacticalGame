## 核心胜利策略

### 1. 双层石墙解读 — 爆破路径规划

**关键决策:** 理解破障行动地图的石墙不是单层而是**双层**(q=-1/q=0/q=1三层)，需要两辆重装分上、下两路逐层爆破，先破左翼再破主墙再破右翼。

```
三层石墙结构（点顶六边形轴向坐标）:
  左翼(q=-1): 爆破点(-1,-1)和(-1,1) — 第2回合
  主墙(q=0):  爆破点(0,-1)和(0,1)  — 第3回合
  右翼(q=1):  爆破点(1,-1)和(1,1)  — 第5回合
  总计: 6次爆破打通两条通道！

爆破时间线:
  Turn 2: 第一轮爆破 → 炸掉左翼(-1,-1)(-1,1)
  Turn 3: 第二轮爆破 → 炸穿主墙(0,-1)(0,1)
  Turn 5: 第三轮爆破 → 突破右翼(1,-1)(1,1)
```

**为什么有效:**
- 侦察兵一开始误判墙厚，T4才发现q=1还有一层墙
- 第5回合立即补爆破，虽然多花2回合，但两辆重装同步推进
- 上、下两路各打通一条通道，互不干扰

### 2. 两路同时破墙 — 上下夹击

**关键决策:** 两辆重装分别负责上、下两路爆破，互不干扰地打开两条进攻通道。

```
破墙任务分配:
  重装#1(上): 爆破(-1,1)→(0,1)→(1,1) → 从(0,1)通道突入
  重装#2(下): 爆破(-1,-1)→(0,-1)→(1,-1) → 从(0,-1)通道突入
  
两条通道贯穿后:
  上路重装最终位置(7,1) — 毗邻HQ
  下路重装最终位置(8,-1) — 毗邻HQ
  两路在敌HQ(8,0)前汇合！
```

**为什么有效:**
- 两辆重装各自执行一条通道，避免了在狭窄缺口处互相阻挡
- 敌方也有两辆重装，被我的双通道吸引分散到两路防守
- 最终两路重装形成包围HQ的钳形攻势

## 失误与改进

### 失误1: 误判墙厚 — 以为只有一层

**问题:** 第3回合以为石墙只有q=0一层，炸开后发现还有q=1的右翼石墙。第4回合重装卡在(0,1)和(0,-1)无法继续东进，浪费了一整回合只能打旁边的敌方单位。

- 爆破(0,1)后重装#1试图穿墙到(2,1)，结果"目标不可达"才发现(1,1)还是墙
- 第5回合才补爆破，多用了2个AP纠正

**改进:** 破障行动的石墙有三层：左翼(q=-1)、主墙(q=0)、右翼(q=1)。在拆第一层时就应该观察地图上的所有blocker分布(特别是q=1行有没有blocker)，一次性规划好爆破路线。

**收益:** 如果T3就猜到第二层墙的存在，两辆重装可以提前部署到(0,1)(0,-1)准备同时爆破，节省1回合 ≈ 早1回合到敌HQ。

### 失误2: 侦察兵路线选择不当

**问题:** 第4回合侦察兵占完cp_sw后，本想转到下方缺口穿墙，但路径被友方步兵(-4,1)阻挡，被迫绕远路到(-5,0)，多花了2回合。

- 侦察兵(-4,3)→(-4,2)被步兵挡路
- 绕到(-5,0)又忘了(-1,0)还有墙
- 最终侦察兵从(-5,0)→(-1,-1)→(0,-1)→(3,-1)用了3回合才穿墙

**改进:** 占点后侦察兵应优先沿着无障碍路径移动，提前规划好穿墙坐标。如果步兵站在关键通道上，应先移动步兵让路。

**收益:** 侦察兵如果能早2回合穿墙，可以用5格移速探查敌方动向，甚至威胁敌方CP。

### 失误3: 未充分利用cp_sw的部署折扣

**问题:** 第5回合占领cp_sw(前线基地，部署折扣-8)后，只在第5回合部署了1个步兵享用了折扣，之后没有再从这个点部署任何单位。

- 步兵45-8=37省8补给 × 1次 = 省8补给
- 对比第26局：从前线基地部署6个单位省了56补给

**改进:** 前线基地是长期杠杆，每多一次部署就多省8补给。剩余3回合内应该持续从cp_sw部署重装或侦查兵。

**收益:** 如果从cp_sw再部署1个重装(92-8=84)或2个侦察兵(38-8=30×2=60)，省16补给搜索+

```
基础收入: 10补给/回合

  第1回合: 50起始

  第4-5回合: 10+8+20=38/回合(3据点)
  第6-8回合: 10+8+20+8=46/回合(3据点, cp_sw)
```

### 2. 攻击机制
```
伤害公式: max(1, 攻击力 - 防御力 + random(-3, 3))
重型兵40攻: 对防御0(HQ)→40±3≈37-43伤
          对重型兵(防13)→27±3≈24-30伤
步兵30攻: 对重型兵(防13)→17±3≈14-20伤
          对侦察兵(防4)→26±3≈23-29伤
```

### 3. 重装爆破机制
```
规则:
  - 只有重型兵可以爆破(无cost，免费动作)
  - 爆破目标必须在距离1内
  - 目标必须是blocker，且不可有单位占据
  - 爆破后该格变plain，可通行
  - 爆破算"act"，与"move"解耦：一个AP可move→demolish或demolish→move

本局爆破记录:
  第2回合: (-1,-1), (-1,1) — 左翼
  第3回合: (0,-1), (0,1) — 主墙
  第5回合: (1,1), (1,-1) — 右翼
  总计: 6次爆破打通两条通道
```

### 4. 破障行动地形关键
```
石墙(blocker, 可爆破):
  左翼(q=-1): (-1,-1), (-1,0), (-1,1) — 左向延伸墙
  主墙(q=0):  (0,-7)到(0,7) — 中央纵向石墙
  右翼(q=1): (1,-6)(1,-5)(1,-4)(1,-1)(1,0)(1,1) — 右向延伸墙

水域(不可通行):
  左上: (-4,-4), (-3,-5)
  右下: (3,5), (4,4)
  右上: (3,-6),(4,-6),(3,-5),(2,-5)
  左下: (-4,6),(-3,6),(-3,5),(-2,5)

三层墙结构:
  进攻方需依次爆破: 左翼→主墙→右翼(或选择单一突破口)
  本局选择两路(上/下)同步推进，各炸3个blocker
```
