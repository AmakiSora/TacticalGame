## 核心教训

### 致命错误: 据点争夺惨败 + HQ 防守真空，经济被碾压后换家竞速输在起跑线

**做了什么:** 前3回合我只抢下1个据点(cp_se 东南补给站)，而对手Hy3-WB抢下3个(cp_nw/cp_w/cp_sw)。到第6回合，对手已占领全部6个据点，经济收入54/回合 vs 我方30/回合。对手用经济优势连续部署7个单位（含3个侦察兵），形成数量与质量双重压制。同时对手重装(8,-1)第6回合已贴到我HQ相邻格，而我方最远的重装(1,2)距对手HQ(-8,0)还有9格——换家竞速完全不可能赢。

**为什么错:** 破障行动地图的胜负手是**据点经济 + 破墙时机 + HQ防守**三位一体。本局我在三个维度全面落后：
1. **据点**: 对手3:1领先，经济差导致对手能持续部署而我只能被动挨打
2. **破墙**: 我第3回合才破墙(0,2)，对手第2回合已破墙(0,-2)并让重装穿墙直插
3. **HQ防守**: 对手重装第6回合已贴脸，我既没有单位封锁HQ邻接格(7,0)/(8,1)/(8,-1)，也没有回防重装驱离

**正确做法:**
- 第1回合应优先抢占**2个据点**（cp_se + cp_e），而非只抢1个
- 第2回合应让重装贴墙(0,2)或(0,-2)准备爆破，与对手同步破墙
- 第5回合发现对手重装逼近HQ时，应立即用侦察兵封锁(7,0)等邻接格，限制对手每回合只能1次攻击

**收益:** 若据点3:3持平，经济差距消失，我可同步推进；若HQ邻接格有人封锁，对手每回合只能打1次(约38伤)，我HQ 100→62→24→可撑到第10回合，为己方重装争取到逼近对手HQ的时间窗。

## 经验教训

### ✅ 做得好的
1. 第3回合成功爆破(0,2)并让重装穿墙插入(0,2)，破墙执行本身无误。
2. 第5~7回合持续攻击敌方贴脸重装(8,-1)，将其从150削到52，输出效率尚可。
3. 第6回合尝试夺回东北前线基地(cp_ne)，虽未成功但意识到位。

### ❌ 致命失误
1. **据点争夺惨败**: 前3回合仅占1个据点(cp_se)，对手占3个。经济差距导致对手能持续部署7个单位、我方仅5个，数量质量双劣。
2. **破墙晚1回合**: 第3回合才破墙，对手第2回合已破墙并让重装穿墙直插——这1回合的时间差让对手尖刀(1,-2)→(3,-2)→...一路滚雪球到(8,-1)。
3. **HQ防守真空**: 第5回合发现敌heavy(7,-1)逼近时，既没有单位封锁HQ邻接格(7,0)/(8,1)/(8,-1)，也没有回防重装驱离。第6回合敌heavy(8,-1)贴脸后，每回合稳定输出约38伤，HQ 63→25→0三回合暴毙。
4. **换家竞速不可能**: 我方最远重装(1,2)距对手HQ(-8,0)有9格，对手重装(8,-1)距我HQ仅1格——同一套"rush HQ"打法，对手抢先6回合到位。

### 🔑 核心教训
> **"据点是经济根基，破墙是进攻前提，HQ是生命线——三者缺一即崩。换家竞速比的是谁先贴到对方HQ，而先手的关键是破墙时机与据点经济。"**
- 前3回合必须拿满己方半场2~3个据点，否则经济差距无法弥补
- 破墙必须在第2回合完成（与对手同步），晚1回合即丧失先机
- 发现敌方单位逼近HQ时，第一优先级是封锁邻接格（每格限制对手1次攻击），而非全力进攻
