- 对手把兵力集中在错误方向（北），我把尖刀送到他最空虚的方向，形成"你重兵在我家门口、我重兵在你老家"的换家错位

### 3. 尖刀重装直插总部（换家竞速抢先到位）

**关键决策:** 破墙后让 heavyS 专职当"尖刀"，沿 (-3,-1)→(-1,-2)→(2,-2)→(5,-2)→(7,-1)→(8,-1) 一路滚雪球直插敌方HQ；第6/7/8回合三记重拳拆光。

### 4. 激活效率：move+attack/demolish 同回合只花1AP

**关键决策:** 利用引擎"首次激活消耗1AP、同回合后续动作免费"的规则，把移动与攻击/拆墙 chaining 在同一点数内。


**为什么有效:**
- 同样5AP，别人只能动5次，我能"动+打"组合出6~7次有效操作
- 拆墙+穿墙同回合只花1AP，是南侧走廊比对手北线突破早打通2回合的关键

### 5. 北盾封堵对手反突破（双向换家对冲）

**关键决策:** 不把双heavy都压上去，留 heavyN 在 (-1,2)/(0,2) 北侧当盾，既对峙对手北线重兵，又持续削对手破墙插入的西进重装。

## 失误与改进

### 失误1: 第5回合 axial 距离算错，浪费一次拆家机会

**问题:** 我误以为 (7,-1) 与 HQ(8,0) 相邻，让重装移动到 (7,-1) 后发动攻击，引擎报 `out of range (2 > 1)`——实际 hex axial 距离 = (|dq|+|dr|+|dq+dr|)/2 = (1+1+2)/2 = **2**，并不相邻。重装已 moved 无法再动，本回合拆家落空，总部多撑了1回合。
**改进:** 每次下达 move/attack 前，先用公式 `dist = (|q1-q2| + |r1-r2| + |(q1+r1)-(q2+r2)|)/2` 校验距离，确认 attack 目标距离 ≤ 单位攻击范围（heavy=1）。正确的拆家位是 (8,-1)（与HQ距离1）。
**收益:** 若第5回合就站上(8,-1)，总部第6回合即降到约23血、第7回合已拆光，可提前1回合结束。**这已是我第二次踩同一坑（#28 T7 也犯过），但本次比#28早2回合察觉并修正，说明距离校验肌肉记忆在形成但尚未固化。**

### 失误2: 部署坐标规划粗糙，一次无效部署

**问题:** 第4回合试图从 cp_nw(-4,-3) 部署 scout@(-3,-2)，因该格与 cp_nw 的 axial 距离=2（非相邻）被拒（out_of_deploy_range），浪费1AP；另第4回合 scout(-4,-2)→(1,-2) 我一度误算距离（早期还有 scoutA 想一步到(1,-2)被拒"not reachable"）。
**改进:** 部署前确认目标格与 HQ/已占据点 **距离=1**（可部署格必须是据点/己方HQ的六角邻居）；移动前确认路径不被己方单位挡死。本局靠即时诊断改格补救，但消耗了若干AP。
**收益:** 每省1个无效AP等于多一次有效推进/输出，对8回合的紧凑进攻节奏影响明显（#28 同样因部署误判浪费AP）。

### 失误3: 斥候过量部署，部分成炮灰

**问题:** 为抢点连续部署多个 scout（T1/T3/T4/T5/T6/T7 共7个），其中 scoutE(4,-3) 终局仅12血被对手削残，部分 scout 投入未转化为持久价值。
**改进:** 抢点用 scout 没问题，但夺点后应换更肉的 infantry 常驻（或让 scout 占完即走、不长期站点）。点占下后要"换防"，把补给更多倾斜给能终结比赛的重装。
**收益:** 减少无谓单位损耗，把补给集中到尖刀与北盾，进攻更锋利。

## 游戏机制深度理解（破障行动地图）

### 1. 经济模型
```
基础收入: 10补给/回合
据点收入: 补给站+20(cp_nw/cp_se)，维修站+8(cp_w/cp_e)，前线基地+8(cp_sw/cp_ne)
本局验证: 1点→18, 2点→38, 3点→46, 4点→54, 5点→62，与读数完全吻合
```

### 2. 行动点 / 单位激活（本局核心效率来源）
```
规则: 单位首次激活(任意 move/attack/demolish)消耗1AP；
      同回合内若已 move（只设 hasMoved，不设 hasActed），再 attack/demolish 免费
      → move+attack = 1AP，move+demolish = 1AP，demolish+move = 1AP
验证: 第3回合 move(-1,-2)+demolish(0,-2)+move(2,-2) 仅 actionsUsed+1；第6回合 move(8,-1)+attackHQ 同理
推论: 每回合5AP可打出"5次激活"，其中若干个可附带1次免费攻击/拆墙，等效6~7次操作
```

### 3. 墙体机制（breach 地图核心）
```
中央石墙: q=0 整列 blocker，仅重装可拆（每格1AP，免费动作）
拆墙条件: 重装与目标墙格距离≤1，且 move 到相邻格后同回合 demolish
本局只拆 (0,-2) 1段——一段足够打通己方半场到敌半场的南侧走廊
东侧(q≥2)无墙，故重装穿墙后一路坦途直插敌HQ
```

### 4. 攻击 / 部署机制
```
伤害 = max(1, 攻击力 − 防御力 + random(-3,3))
heavy 攻击力40、防御13、移动3、攻击范围1、造价92
HQ: 100HP，0防御 → heavy 每击约37~43伤，3击摧毁
部署: 任意己方据点/HQ 1格内可部署；前线基地范围内-8折扣；部署单位当回合不可移动、可攻击
```

### 5. 据点 / 胜负判定（最关键的坑，源自#29教训）
```
占领: 回合结束时 canCapture 单位(scout/infantry)站在点上即易主；heavy canCapture=false（占位≠占领）
胜负: 敌方 HQ hp 归零 → combat.ts 立即 endGame(owner,'headquarters_destroyed') → 瞬杀判负
      adjudicationWeights 总分仅在双方 HQ 都活到第15回合时才用于裁定
本局应用: 我作为进攻方，利用"HQ瞬杀"规则在T8抢先拆光对手HQ；对手南翼空虚、来不及回防，换家竞速我胜
```

**结论 — 三局串起来的规律：**
1. **破障行动存在两种胜利 archetype**：评分流（稳健、打满）与强攻流（拆HQ、速胜）。当对手 HQ 防守空虚、我先手抢到突破节奏时，强攻流远快于评分流。
2. **#29 的惨败教训（HQ瞬杀、残血HQ优先于进攻）在本局被反向应用**：我作为进攻方，正是利用对手"南翼空虚、重兵堆北"的布阵，抢在对手回防前拆光其HQ——把#29里害我输的规则，变成了我赢的武器。
3. **axial 距离误算是我反复发作的弱点**（#28 T7、#30 T5 同坑），但本次比#28早2回合察觉修正，节奏因此更快（8回合 vs 10回合）。距离校验必须固化为每次 move/attack 前的强制步骤。
4. **经济先行是共通的本体**：三局都验证"前3回合抢满己方半场据点"是基本盘；本局5:1的据点差，让即便拖入评分我也稳赢，强攻只是更快的保险。

3. **尖刀直插**: heavyS 专职滚雪球到(8,-1)，第6~8回合三击拆光HQ，比对手回防快约3步
4. **经济碾压**: 5:1 据点差锁62/回合收入，支撑7次部署与持续强推
5. **北盾封堵**: heavyN 钉死对手越境重装，使我只需单向rush、无后顾之忧

### 核心战术原则

> **"看穿对手空虚侧，破墙开血路，尖刀直插拆总部——换家竞速，谁先贴脸谁赢。"**

- 据点是本体：收入、部署锚点、治疗全在点上，前3回合必须拿满己方半场
- 进攻方要自己造突破口：重装 move+demolish 同回合只花1AP，早拆早通；选在对手空虚的一侧开洞
- 每台重装"动+打"组合省1AP，8回合等效多打2~3个动作，节奏就是生命
- 拆家只要求尖刀"存活且站邻格"不要求满血——算清对手1回合集火打不死，就敢顶着反扑补刀
- **每次 move/attack 前强制用 axial 距离公式校验**，杜绝"以为相邻其实距离2"的低级失误

### 一句话总结

**破障行动第30局，我看穿对手把重兵堆在北侧、南翼洞开的布阵，第3回合在空虚的南侧一锤凿穿石墙，让尖刀重装顺着它直插敌方HQ脸前，第6~8回合三记重拳把100血总部砸成0——胜负从破墙、更从"读穿对手阵型弱点"那一刻就写好了。**
