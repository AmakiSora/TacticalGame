## 核心胜利策略

### 1. 经济优先：抢占cp_e 补给站

**关键决策:** 第1轮即派侦察兵从(7,1)全速5格抢占 cp_e(3,0)，回合结束时自动占领。

```text
第1轮 / player_a: 侦察兵(7,1)→(3,0)，第1轮结束自动占领 cp_e（补给型，+8收入/回合）
第1轮 / player_b: 对手占 cp_nw(+8)
第1轮 / player_c: 对手占 cp_sw(+8)
```

**为什么有效:**
- cp_e(3,0) 距我方HQ(8,0)仅5格，侦察兵移动力5恰好到达
- 占领后每回合+8供给，使我方经济从基础6提升至14（含cp_e 8），为部署提供持续资金
- 后期前线基地折扣部署进一步降低新兵成本

### 2. 前沿部署：cp_ne 成为进攻跳板

**关键决策:** 第2轮侦察兵(3,0)→(3,-3)占领 cp_ne（前线基地，部署折扣-4）。

```text
第2轮 / player_a: 侦察兵占领 cp_ne，此后所有从此点部署的单位享受-4供给折扣
```

**为什么有效:**
- cp_ne 距敌方单位更近，部署的新单位可立即投入战斗
- 折扣使侦察兵成本从38降至34，步兵从45降至41
- 多次从此点部署，累计节省约20供给

### 3. 多路夹击：集中火力消灭核心单位

**关键决策:** 第2-5轮间，步兵(4,0)→(1,0)→(3,-2)夹击侦察兵_b(2,-2)，配合侦察兵(3,-3)残血补刀。

```text
第2轮 / player_a: 侦察兵(3,0)→(3,-3)、步兵(4,0)→(1,0) 
第3轮 / player_a: 步兵(3,2)攻击(2,-2)、侦察兵(3,-3)夹击、步兵(1,0)攻击(0,0)
第5轮 / player_a: 合计消灭步兵_b(77→0)、侦察兵_b(26→0)
第10轮 / player_a: 合计消灭侦察兵_b(-1,-4)(65→0)、步兵_c(75→52)
```

**为什么有效:**
- 多个单位从不同方向逼近同一目标，迫使敌方无法全部防守
- 残血单位由不同友军补刀，最大化杀伤效率
- 消灭关键单位后敌方前线出现空洞，为我方推进创造空间

### 4. 消耗战拖垮对手AI响应

**关键决策:** 第6-12轮保持持续压力，不断在多个前线发动攻击，使对手AI（尤其是切换为agnes2.0flash后）响应频率大幅降低。

```text
第6-12轮: 共发动20+次攻击行动，每轮同时推进多方向
效果: player_b出现多次10+分钟零行动、player_c单位逐步被消耗
```

**为什么有效:**
- 多人局中一方AI卡顿后，另一方获得实际2v1优势
- 持续骚扰使对手无法形成有效反击
- 经济优势（5 CP controled vs 1-3）转化为部署速度差距

### 5. 第五轮歼灭战锁定优势

**关键决策:** 第5轮集中所有可用行动点集火消灭 player_b 的步兵(1,-1)和侦察兵(2,-2)。

```text
第5轮 / player_a: 步兵(3,-1)攻击(1,-1), 侦察兵(3,-2)攻击(1,-1)→击灭
              侦察兵(2,-2)攻击(2,-2), 侦察兵(0,-3)夹击→击灭
```

**为什么有效:**
- 一举消灭 player_b 前期的两个核心单位，使其短期无法组织有效反击
- player_b 此后几乎无新生力量补充，最终只保留 1 个据点

## 失误与改进

### 失误1: 未更早占领cp_center

**问题:** 第1-3轮专注于清理对方残兵和架设前线基地，未及时在步兵(4,0)前占位(0,0)。直到第9轮才短暂占领，但此后又因对手反扑失去。中期中央控制权多次易手，浪费了大量行动点反复拉锯。

**改进:** 第2轮占cp_ne后，应立即派步兵(1,0)或(3,2)转向(0,0)占位。更佳方案是在第3轮步兵(4,0)→(1,0)后下一步迈向(0,0)，而非继续清理敌方偏远单位。

**预期收益:** 若第4轮即控制cp_center，将减少至少3轮中央拉锯，节省6+行动点用于扩张或攻击。

### 失误2: 行动点分配不够高效

**问题:** 多轮出现"已激活单位推到位但无法攻击目标"的情况。例如第6轮试图将步兵(5,-1)推到(2,-1)攻击敌步兵(1,-1)，但因地图中央cluster blockers（地形障碍六边形）阻挡无法到达，浪费行动点。

**改进:** 部署前应预判下一步行动路径是否 open，优先选择障碍物少的路线推进。可利用重型单位破坏地形开路（但本局未部署重型单位）。

**预期收益:** 每浪费1个行动点相当于损失约22-30伤害值，整局若减少5次低效移动可多击杀1-2个敌方单位。

### 失误3: 未充分使用集中部署策略

**问题:** 供给充足时多轮未及时将供给转化为战斗力。第6-9轮供给接近50-70时仅作简单攻击而未部署新兵，导致大量供给未投入战场，提供的裁决分数也有限。

**改进:** 当供给高于50时应立即规划部署行动，利用前线基地折扣补充新兵。尤其在我方控制cp_w、cp_ne、cp_e三个基地时可形成向心部署态势。

**预期收益:** 每轮多部署1个侦察兵（34供给），配合前线推进，可额外获得约60-80点的存活军力价值（按 armyValue=1 计算）。

## 总结

### 胜利关键因素
1. 经济优先：首回合即占cp_e，建立持续供给优势
2. 多路夹击：3-5轮通过协同攻击消灭敌方核心战力
3. 前线渗透：利用cp_ne/cp_w实现低成本快速部署
4. 高压消耗：持续骚扰迫使其余两方无法反击，最终裁判分占优

### 核心战术原则
> **"先据要地、多路推进、集中火力、持续施压"**

### 一句话总结
**通过经济部署协同多路夹击，将战术优势累积为裁判分数优势，在15轮整轮限制下以第一名完赛。**
