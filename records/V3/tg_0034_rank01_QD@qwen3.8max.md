## 核心胜利策略

### 1. 双控制点经济碾压 — cp_w + cp_sw

**关键决策:** 第8回合占领cp_w（西部前线基地），第10回合夺回cp_sw（西南补给站），形成双点经济体系。

```
Turn 8: 侦察兵2(-4,0)→(-3,0) 占领cp_w — 收入+4/轮，部署折扣4
Turn 10: 新侦察兵(-4,4)→(-3,3) 占领cp_sw — 收入+8/轮
合计收入: 6基础 + 4(cp_w) + 8(cp_sw) = 18/轮
```

**为什么有效:**
- 2控制点 = 200判决分（其他玩家最多1个=100分），直接领先100分
- 18/轮收入 vs 其他玩家最多14/轮，后期补给积累更快
- cp_w的部署折扣4让步兵从45降到41，每次部署多赚4分

### 2. 集火消耗战术 — 逐步歼灭player_c

**关键决策:** 第6-10回合集中全部火力逐步消灭player_c的侦察兵群，从5个单位削减到0。

```
Turn 6: 3单位集火击杀侦察兵(-4,2) [65HP→0]
Turn 7: 步兵+侦察兵攻击(-3,0)伤~38 [65→~27HP]
Turn 8: 步兵击杀(-3,0) [7HP→0]，侦察兵1攻击(-4,1)伤12
Turn 9: 侦察兵1+2集火(-4,1)伤~24 [28→~4HP]
Turn 10: 侦察兵1击杀(-4,1) [6HP→0]
```

**为什么有效:**
- 消灭player_c全部军队 = 其军队价值归零（最终player_c得分仅308，垫底）
- 每次集火确保击杀，不分散火力
- 消灭邻近威胁后才能安全占领控制点

### 3. 部署得分机制 — 最后两轮的发现

**关键决策:** 第14-15轮发现"部署单位=净增判决分"，因为军队价值(maxHP) > 部署成本。

```
步兵: 花费45(折扣后41) → 军队价值+100 → 净增+59分！
侦察兵: 花费38 → 军队价值+65 → 净增+27分！
游骑兵: 花费72 → 军队价值+72 → 净增0（不值得）
支援: 花费60 → 军队价值+82 → 净增+22分

Turn 14: 部署步兵cp_w(41) + 部署侦察兵cp_sw(38) = +86分
Turn 15: 部署步兵cp_w(41) = +59分
两轮合计: +145分！
```

**为什么有效:**
- 判决公式中armyValue×1和supplies×1权重相同
- 部署单位将"补给"转化为"军队价值"，如果军队价值>成本就是净赚
- 步兵是最高效的部署选择（净增59分），远超侦察兵（27分）
- 这145分是我最终领先15分的关键！如果没有这两轮部署，我会以476:606输给player_d

### 4. 战术撤退保存实力 — 第2-3轮的正确决策

**关键决策:** 面对player_c的早期进攻，选择撤退而非硬拼，保存全部5个单位。

## 失误与改进

### 失误1: 六边形距离计算错误（第5回合）

**问题:** 错误认为(-4,3)、(-3,2)、(-5,3)与(-4,1)相邻（实际距离均为2），浪费3个行动点。
**改进:** 牢记相邻格公式：(q,r)的6个相邻格为(q+1,r),(q+1,r-1),(q,r-1),(q-1,r),(q-1,r+1),(q,r+1)。攻击前必须心算验证距离。
**收益:** 避免浪费行动点，第5轮本可以多造成30+伤害。

### 失误2: 第11-13轮过于保守，未利用部署得分

**问题:** 第11轮有50补给、第12轮有68补给、第13轮有86补给，完全可以部署步兵(41)获得+59分，但我选择了"防守保分"直接结束回合。
**改进:** 只要补给≥41且有cp_w，就应该每轮部署步兵！部署=净增59分，没有任何风险。
**收益:** 如果第11-13轮各部署1个步兵，可多获得3×59=177分，总分将达到798，远超第2名。

### 失误3: 第9轮侦察兵2离开cp_w

**问题:** 第9轮让侦察兵2从cp_w(-3,0)移到(-4,0)去攻击敌人，导致cp_w无人驻守。虽然回合结束时仍是我占领（敌人没来夺），但存在风险。
**改进:** 控制点上的单位不应轻易离开，除非确定当轮不会有敌人来占。应该用其他单位去攻击。
**收益:** 避免被偷点的风险（被偷=损失100判决分+4收入/轮）。

### 失误4: 早期未尝试部署得分

**问题:** 整局游戏直到第14轮才发现"部署单位=净增分"的机制。如果从第8轮（获得cp_w后）就开始每轮部署步兵，可以积累更多分数。
**改进:** 获得forward_base后，每轮优先部署步兵（净增59分），这是无风险操作。
**收益:** 第8-13轮共6轮×59=354分潜在收益，即使只执行一半也能让胜利更加稳固。

## 游戏机制深度理解

### 1. 经济模型
```
基础收入: 6补给/回合
控制点收入: supply=8, forward_base=4, repair=2
起始补给: 70
行动点: 7/回合
Comeback机制: 第3轮起，分差>40%时落后方+20/轮
```

### 2. 攻击机制
```
伤害公式: max(1, 攻击力 - 防御力 + random(-3, 3))
步兵30攻: 对侦察兵(防2)≈28伤, 对步兵(防4)≈26伤
侦察兵16攻: 对侦察兵(防2)≈14伤, 对步兵(防4)≈12伤
游骑兵44攻(射程3): 对步兵≈40伤, 对侦察兵≈42伤
```

### 3. 评分机制（判决权重）
```
总分 = 敌HQ伤害×5 + 我方HQ_HP×1 + 控制点×100 + 军队价值×1 + 补给×1
军队价值 = 所有存活单位的maxHP之和（不是当前HP！）
本局: 0×5 + 180×1 + 2×100 + 239×1 + 2×1 = 621
```

### 4. 部署得分机制（关键发现！）
```
部署单位: 花费补给 → 获得军队价值(maxHP)
步兵: 花45(折扣41) → +100军队 → 净增+59分 ⭐最高效
侦察兵: 花38 → +65军队 → 净增+27分
支援: 花60 → +82军队 → 净增+22分
游骑兵: 花72 → +72军队 → 净增0（不值得！）
结论: 只要补给≥41且有forward_base，就应部署步兵！
```

### 5. 地形关键
```
Blockers(不可通行): (2,-1), (1,1), (-1,2), (-2,1), (-1,-1), (1,-2)
特点: 6个blocker形成中心环状，将地图分为内外两圈
关键影响:
  - (-2,1)blocker阻断了从cp_sw(-3,3)直接北上到cp_w(-3,0)的路线
  - (-1,2)blocker阻断了从南侧接近cp_se(0,3)的路线
  - 从我的区域到中央必须绕行
```

## 总结

### 胜利关键因素
1. **🎯 双控制点(200分)** — 第8、10轮先后占领cp_w和cp_sw，锁定200分优势
2. **⚔️ 集火消灭player_c** — 5个击杀让邻近对手军队归零，解除侧翼威胁
3. **💡 部署得分发现** — 最后两轮部署步兵+118分，是15分险胜的决定性因素
4. **🛡️ 早期战术撤退** — 第2-3轮保存实力，避免被player_c早期压制歼灭
5. **🏰 HQ从未受损** — 180满血=180分稳定贡献

### 核心战术原则

> **"占点为王、集火歼敌、部署增分、撤退保命"**

- 控制点×100是最大得分来源，优先占领并死守
- 多人局中消灭邻近对手的军队，比攻击 distant 敌人更有效
- 部署步兵(净增59分)是无风险得分手段，有forward_base就应每轮部署
- 面对优势敌人时战术撤退不丢人，保存军队价值就是保存分数
- 六边形距离必须精确计算，一个错误浪费一个行动点

### 一句话总结

**第34局以双控制点(200分)和最后两轮部署得分(+118分)险胜player_d仅15分，证明在六方环线6人混战中"控制点+军队数量"是判决核心，而"部署步兵=净增59分"是此前从未被发现的关键得分机制。**
