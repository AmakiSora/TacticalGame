## 核心胜利策略

### 1. 开局抢占经济点，建立补给优势

**关键决策:** 第1轮先手立即占领`cp_supply_a`(supply+12)，确保稳定经济来源。

```text
第1轮 / player_b: 步兵(-5,4)→(-4,2)，回合结束占领cp_supply_a
第2轮 / player_b: 获得收入12(补给)+10(基础)=22补给
第3轮 / player_b: 占领cp_fb_a(+8)后，每轮收入12+8+10=30补给
```

**为什么有效:**
- 补给型据点提供12补给/轮，高于普通据点的10，且本局没有其他更高收入来源
- 3个据点(含supply+fb_a+repair)合计收入30/轮，足以支撑每轮部署1个单位
- 第4轮起持续保持经济优势，最终累计收入远超对手

### 2. 中央开路，打通战略通道

**关键决策:** 第3轮用重装兵拆除障碍(-2,1)，打开中央通路。

```text
第3轮 / player_b: 重装兵(-2,3)→(-2,2)，拆除障碍(-2,1)
效果: 打通中央通道，使后续部队可以南北迂回
```

**为什么有效:**
- 熔炉重铸地图中央的障碍群（(-1,0)、(0,0)、(1,0)、(0,-1)、(0,1)、(2,-1)、(-2,1)）将地图分割为南北两路
- 拆除(-2,1)后，我方部队可以从西侧直接进入中央区域，缩短了到敌方控制点的距离
- 后续侦察兵和游骑兵得以通过此通道进入敌方半场发动攻击

### 3. 集火击杀，逐步取得人数优势

**关键决策:** 第4-6轮对敌方前锋步兵持续集火，最终在第6轮击杀。

```text
第4轮 / player_b: 步兵(-1,-1)攻击敌方步兵(0,-2) → 90→67HP
第5轮 / player_b: 步兵(-1,-1)攻击 → 67→47HP，新步兵(-1,-2)攻击 → 47→29HP
第6轮 / player_b: 步兵(-1,-1)攻击 → 29→11HP，新步兵(-1,-2)攻击 → 11→0HP **击杀！**
```

**为什么有效:**
- 连续3轮集火同一目标，累计造成90伤害，正好击杀步兵(90HP)
- 击杀后获得6:5人数优势，从此占据战场主动权
- 第8轮再次集火击杀敌方游骑兵，进一步巩固优势

### 4. 支援单位救场，维持前线战力

**关键决策:** 第5-8轮支援单位持续为前线步兵提供治疗，防止被击杀。

```text
第5轮 / player_b: 支援(-3,2)→(-2,1)
第7轮 / player_b: 支援(-2,-1)治疗步兵(-1,-1) 10→35HP
第8轮 / player_b: 支援治疗步兵(-1,-1) 3→26HP
```

**为什么有效:**
- 两次治疗分别从10→35和3→26，共挽救约48HP
- 虽然第8轮末步兵仍被击杀，但多争取了2轮输出时间
- 支援单位放置在(-2,-1)位置，可同时覆盖中央和南部前线

### 5. 终局部署重装兵，最大化军力评分

**关键决策:** 第10轮（最后一轮）从`cp_fb_a`部署重装兵，消耗80补给（折扣后）。

```
成本: 88-8(折扣)=80补给
军力价值: 88 × 2(权重) = 176分
净收益: 176 - 80(补给损失) = 96分
```

**为什么有效:**
- 裁决权重中`armyValue`为2，即每点军力价值贡献2分
- 重装兵(88军力)贡献176分，仅消耗80补给（损失80分）
- 净收益约96分，最终分差179分，此部署贡献了约一半的优势

## 失误与改进

### 失误1: 第7轮未及时后撤濒死步兵

**问题:** 第7轮开始时，步兵(-1,-1)仅10HP，步兵(-1,-2)仅60HP。虽然支援治疗了(-1,-1)至35HP，但未将两个步兵后撤。第8轮敌方重装兵和游骑兵联手击杀两个步兵，导致第8-9轮兵力劣势。

**改进:** 当步兵HP低于30时，应后撤至安全位置（如(-2,-1)或(-3,-2)），让支援在前线吸引火力，或用新部署单位替换。

**预期收益:** 避免2个步兵被击杀，节省42+42=84补给部署成本，并保持连续6轮的人数优势。

### 失误2: 第5轮未在集火同时部署防守单位

**问题:** 第5轮消耗4个行动点全部用于进攻和支援移动，未部署新单位。敌方第7轮部署了新步兵，导致第8轮6:6兵力平局后被反杀。

**改进:** 第5轮用1个行动点部署步兵从`cp_fb_a`(-2,-2)至(-1,-2)或(-3,-1)，增加前线兵力密度。

**预期收益:** 第8轮时保持7:6或7:7兵力，避免被反杀。

### 失误3: 第9轮后未拆除更多障碍

**问题:** 仅拆除了一个障碍(-2,1)。中央还有多个障碍((0,0)、(1,0)、(0,-1)、(0,1))限制通行。如果多拆除1-2个障碍，可以开辟更多进攻路线。

**改进:** 第7-9轮重装兵在(-1,1)时可以移动到(0,1)或(0,0)附近拆除障碍，打开通往敌方半场的直接通道。

**预期收益:** 更多进攻路线，可对敌方HQ施加压力，争取HQ伤害分(4×权重)。
