## 核心胜利策略

### 1. 首轮对称占点，确保重装兵准时上线

**关键决策:** 第1轮用侦察兵直接落在东线补给点 `(3,-1)`，把后续收入提高到14/轮。

```text
第1轮 / player_a: 侦察兵(-5,4)→西线补给点(-3,1)，占点（seq4-5）
第1轮 / player_b: 侦察兵(5,-4)→东线补给点(3,-1)，占点（seq9-10）
第2-3轮 / player_b: 每轮获得6基础+8据点=14补给（seq17、26）
第3轮 / player_b: 补给54，支付50部署重装兵（seq28）
```

**为什么有效:**

- 初始补给20，仅靠基础收入6要到第6次收入后才能部署50费重装兵；首轮占点后，第3轮已有54补给。
- 本图只有1行动点，经济并非为了多单位铺场，而是为了尽早把唯一关键单位重装兵送上场。
- 东线点始终由我方控制，最终仍贡献1个据点和每轮8补给。

### 2. 部署后立即攻击，吃满单次激活价值

**关键决策:** 第3轮在 `(2,1)` 部署重装兵后，利用 `hasActed=false` 立即攻击相邻敌侦察兵。

```text
部署: heavy → (2,1)，花费50，actionsUsed=1（seq28）
后续攻击: heavy → scout(1,2)，造成33伤害，actionsUsed仍为1（seq29）
```

**为什么有效:**

- `actionsPerTurn=1` 时，同一已激活单位的合法后续动作是最稀缺的节奏来源。
- 这次免费攻击把敌侦察兵压到32HP；第4轮一次攻击即可击毁，最终我方单位击杀数为1。
- 同一机制在第5轮表现为“爆破后移动”，第6轮表现为“移动后爆破”，第7轮表现为“移动后攻击”。

### 3. 选择北侧双爆破链，先一步进入HQ射程

**关键决策:** 从 `(2,1)` 沿 `(1,1)→(-1,1)→(-2,1)` 推进，爆破 `(1,1)` 与 `(-2,1)`。

```text
第5轮: 拆(1,1)，heavy进入(1,1)（seq49-50）
第6轮: heavy进入(-1,1)，拆(-2,1)（seq60-61）
第7轮: heavy进入(-2,1)，攻击HQ(-2,0)（seq71-72）
```

**为什么有效:**

- `(-2,1)` 与敌HQ `(-2,0)` 的六角距离为1，是稳定输出位。
- `player_a` 的南侧路线需要连续处理 `(-1,-1)`、`(-1,0)`、`(1,0)`，直到第8轮才从 `(1,0)` 首次攻击我方HQ。
- 我方第7轮已打出35伤害，形成一整次重装攻击的领先。

### 4. 用确定伤害区间锁定攻城竞速

**关键决策:** 双方重装兵贴HQ后，不分兵、不换目标，连续攻击敌HQ。

```text
我方HQ攻击: 35、33、37、39事件伤害；有效HQ伤害120
对手HQ攻击: 38、34、35；有效HQ伤害107
第10轮行动前: 敌HQ 15HP，我方HQ 13HP
```

**为什么有效:**

- 重装兵攻击40、HQ防御4、伤害浮动±3，每击为33-39伤害。
- 120HP的HQ至少需要4次重装攻击；三次最高仅117，无法击毁。
- 我方先命中一次，所以第10轮即使对手先打出第三击，我方也必定存活并完成第四击。

## 失误与改进

### 失误1: 第4轮补刀侦察兵，延后了一轮开路

**问题:** 第3轮已把敌侦察兵打到32HP，但第4轮仍花掉唯一行动点完成击杀（seq38-39）。该侦察兵攻击16，而我方重装防御13、150HP，对重装兵只能造成低伤害；即使它转攻HQ，也必须占用 `player_a` 自己的唯一行动点。实际选择使首次HQ攻击发生在第7轮。

**改进:** 第4轮直接让重装兵拆 `(1,1)` 并进入该格，把侦察兵视为可接受的次要威胁；仅当它能当轮击毁HQ、翻掉关键据点或封死重装路径时才补刀。

**预期收益:** 北侧爆破链可整体提前约1轮，理论上第6轮开始攻击HQ，把最终13HP的险胜变为至少多承受一击的安全优势。该估算假设敌侦察兵不成功封堵爆破路径。

### 失误2: 第2轮外圈侦察移动没有形成后续价值

**问题:** 侦察兵 `(5,-5)→(2,-5)`（seq19）后直到游戏结束都没有再次产生移动、攻击或占点事件。虽然当轮尚不足50补给、没有耽误部署重装兵，但这次位置投资没有转化为HQ伤害、据点或防守。

**改进:** 第2轮应先根据对手侦察的真实路线选择更有条件性的落点：若其继续冲HQ，则让侦察靠近己方入口做卡位；若其留守西点，则沿外圈明确规划两步威胁西点。没有后续激活计划时，不应把“前推”本身当作收益。

**预期收益:** 不保证直接增加分数，但能让唯一一次空闲行动形成可执行的第二威胁，减少全局完全依赖单一重装兵的风险。

### 失误3: 第6轮曾误判HQ六角距离

**问题:** 实时操作中曾把 `(-1,1)` 到敌HQ `(-2,0)` 误判为距离1，攻击请求被API以距离2拒绝。回放中没有生成失败攻击事件；seq60之后直接是合法爆破seq61，因此补给、行动点和棋面均未受影响。

**改进:** 每次攻击前显式计算 `max(|dq|, |dr|, |ds|)`，其中 `s=-q-r`；对本例差值为 `dq=-1, dr=-1, ds=2`，距离应为2。

**预期收益:** 避免实时限时环境中的决策延迟，并防止把错误距离用于不可逆移动规划。

## 总结

### 胜利关键因素

1. 第1轮占领东线补给点，使第3轮能准时以54补给部署重装兵。
2. 四次关键复合动作都只消耗1行动点，最大化了 `actionsPerTurn=1` 的单位激活价值。
3. 北侧双爆破路线在第7轮先命中敌HQ，比对手首次HQ攻击早一轮。
4. 用33-39的确定伤害区间判断竞速，不在最后三轮被低价值目标分散。
5. 最终以13HP存活，完成120点有效HQ伤害并淘汰 `player_a`。

### 核心战术原则

> **“一行动图先抢收入出重装，复合动作开最短墙线，先贴HQ后只算斩杀线。”**

### 一句话总结

**我方以后手首轮占点、第三轮重装上线和北侧双爆破链抢到一次HQ攻击领先，最终在双方HQ仅剩15与13HP时完成第四击，以13HP赢下危险距离攻城竞速。**
