## 核心胜利策略

### 1. 抢先抢占cp_2供给据点

**关键决策:** 第1-3轮集中侦察兵推进至cp_2，比对手更快完成据点占领。

```text
第1轮 / `player_b`: 侦察兵`504cd19b` (5,-4)→(4,-3)→(3,-2)
第3轮 / `player_b`: 侦察兵`504cd19b` (3,-2)→(3,0) 站上cp_2
第3轮 / 回合结束: 据点cp_2被我方捕获
```

**为什么有效:**
- cp_2是supply类型据点，提供+8/回合收入
- 我方从第3回合起获得14/回合（基础6+据点8），而对手仍只有6/回合
- 经济领先使我在第8轮就能部署重型，而对手到第8轮才部署步兵

### 2. 重型突破战术 — 拆墙直达敌方HQ

**关键决策:** 攒够50补给后第一时间部署重型，利用heavy的demolish能力拆除阻挡通路的关键墙体。

```text
第8轮 / `player_b`: 从cp_2部署重型`8ca8d65e`至(4,0)，花费50
第9轮 / `player_b`: 重型 (4,0)→(4,1)→(3,1)→(2,1)→(1,1) 逐步推进
第12轮 / `player_b`: 重型抵达(1,1)，已深入敌阵腹地
第18轮 / `player_b`: 重型拆除(2,-1)墙体
第21轮 / `player_b`: 重型拆除(-1,0)墙体，直通敌方HQ
```

**为什么有效:**
- 地图"危险距离"的核心设计就是1行动点/回合的慢节奏攻城
- heavy的attack=40, defense=13，是摧毁HQ的最高效单位
- 拆除(-1,0)是关键转折——这是敌方HQ(-2,0)与外界之间的最后一道屏障

### 3. 单重型持续攻击 — 以质取胜

**关键决策:** 不分散兵力，集中所有资源培养一个主力重型作为攻城锤。

```text
第22轮 / `player_b`: 重型`8ca8d65e` (-1,0)→攻击HQ，伤害37，HQ 120→83
第23轮 / `player_b`: 重型`8ca8d65e` (-1,0)→攻击HQ，伤害37，HQ 83→49
第24轮 / `player_b`: 重型`8ca8d65e` (-1,0)→攻击HQ，伤害34，HQ 49→15

```

**为什么有效:**
- 重型attack=40，HQ defense=4，基础伤害36，方差±3
- 每次攻击实际造成34-37伤害，4次攻击总计120恰好摧毁HQ
- 对手虽也部署了5个单位（含3个重型），但全部用于防守，未能对我方HQ造成任何伤害

## 失误与改进

### 失误1: 前期侦察兵推进过于分散

**问题:** 第4轮部署的第二个侦察兵`34f7c67e`从(4,0)→(3,-1)→(3,-3)绕了很大一圈，走了很多弯路才回到前线轴线。如果直接部署重型会更高效。

**改进:** 在已有cp_2收入保障的情况下，应更早（第6-7轮）就部署重型而非侦察兵，减少无意义的走位。

**预期收益:** 提前2-3轮完成突破，可能将总回合数从25轮缩减至22轮左右。

### 失误2: 重型多次往返调整路线

**问题:** 重型`8ca8d65e`从(0,1)→(1,-1)→(1,0)→(0,1)→(1,-1)→(1,0)来回折腾了3-4轮才找到正确的拆墙路线。这说明对地图障碍物的路径规划不够精确。

**改进:** 在行动前应先在脑海中绘制出从(4,0)到(-2,0)的最短拆墙路径，一次性规划好每一步。

**预期收益:** 节省3-4个回合的行动点，转化为3-4次额外攻击。

### 失误3: 过度部署单位

**问题:** 我方共部署了6个单位（1侦察兵+5重型），花费258补给。但真正发挥作用的只有1个重型（`8ca8d65e`），其余5个单位几乎全程闲置。

**改进:** 应集中资源只部署1-2个重型，将节省的补给留作最终冲刺时的缓冲。

**预期收益:** 减少约150补给浪费，最终补给可从102提升至约140+。
