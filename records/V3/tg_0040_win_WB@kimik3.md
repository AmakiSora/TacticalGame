## 核心胜利策略

### 1. 后手破对称：南线闪击制造非对称压力

**关键决策:** 开局不与对手镜像发育，侦察兵`bca14700`沿南线 `(-5,4)→(-3,5)→(0,3)→(2,1)` 全速直扑敌HQ，第3轮即开始削血。

```text
第1轮 / player_a: 侦察(-5,4)→(-3,5)（seq8）
第2轮 / player_a: 侦察(-3,5)→(0,3)（seq18）
第3轮 / player_a: 侦察(0,3)绕开(1,2)拦截→(2,1)，攻敌HQ 11伤（seq28-29）
```

**为什么有效:**
- 本图双方HQ同距4格、完全180°旋转对称，`player_b` 先手。任何镜像竞速都是先手方胜——后手唯一的胜机是**打破对称**。
- 侦察兵闪击走南线 `(-3,5)→(0,3)→(2,1)` 仅需3次移动即达敌HQ邻格，且 `(2,1)` 是"堡垒位"：其开放邻格 `(3,1)/(2,2)/(1,2)` 全在我方半场方向，敌单位难以贴身驱赶。
- 对手被迫在第3轮把占点侦察从 `(3,0)` 调回 `(1,2)` 拦截（seq23），经济部队被迫参战。

### 2. 诱饵换时间：一只侦察兵钉死敌方重装2个回合

**关键决策:** 第4轮侦察兵不逃，继续攻击敌HQ（14伤），明示"持续威胁"，诱使对手用重装兵猎杀。

```text
第4轮 / player_a: 侦察攻敌HQ 14伤（seq38）
第5轮 / player_b: 重装(3,-1)→(3,0) 攻侦察 36伤（seq43-44）← 咬钩
第6轮 / player_b: 重装攻侦察 33伤，击杀（seq53）
```

**为什么有效:**
- 1AP地图上行动点就是一切。对手为重装付出的50补给，其预期用途是爆破；被我38补给的侦察兵牵走 **2个完整席位回合**（移动+攻击、原地击杀），其走廊进度直接落后2轮。
- 同期我方HQ承伤为0：猎杀发生在地图对角，我方防线毫无压力。
- 这是本局的分水岭：第6轮我方爆破 `(-1,-1)` 时，对手重装还在 `(3,0)` 擦刀；等它第7轮回到 `(2,1)` 爆破 `(1,1)`，镜像竞速的时钟已被偷走2轮。

### 3. 最短爆破链：(-1,-1)→(1,0) 两次爆破，移动+爆破复合

**关键决策:** 重装从 `(-2,-1)` 出发，只拆通向敌HQ `(2,0)` 的中缝两块墙，配合移动+爆破同回合复合，第8轮开门、第9轮贴脸。

```text
第6轮: 爆破(-1,-1)（seq58）
第7轮: (-2,-1)→(0,-1)（seq68）
第8轮: (0,-1)→(1,-1)，爆破(1,0)（seq77-78）
第9轮: (1,-1)→(1,0)，攻敌HQ 34伤（seq87-88）
```

**为什么有效:**
- 危险距离中央是空心墙，口袋区 `{(0,0),(0,±1),(1,-1),(1,-2),(-1,1),(-1,2)}` 与外围完全隔离，**唯一正攻是重装爆破**；`(-1,-1)+(1,0)` 是我侧到敌HQ的最短链（2拆3走）。
- 对手选链 `(1,1)→(-1,1)→(-1,0)/(-2,1)` 明显更长：其重装第9轮才到 `(-1,1)`，第11轮才拆第二块墙 `(-2,1)`，距离首次打我家至少还需2轮（理论最早第12-13轮）。
- 竞速结果：我方第9轮首刀，对手理论首刀第12轮——**3轮差距即胜负**。

### 4. 读懂裁决权重：supplies=0，经济优势是纸面富贵

**关键决策:** 全程不与对手争夺 `cp_2`、不回访经济，把所有行动点押在HQ伤害上。

```text

我方账本: 收入66(6×11)、终局补给36 → 同样计0分
裁决结果: HQ伤害 120×20=2400 对 0×20=0
```

**为什么有效:**
- 本局 `adjudicationWeights.supplies=0`、`enemyHqDamage=20`——囤积补给零价值，HQ伤害是唯一大头。
- 对手 `cp_2` 带来66点额外净收入（132 vs 66），但换成一句残酷的话：**没变成HQ伤害的收入等于没有收入**。
- 对照 tg_0038 结论"钱的唯一意义是尽早把重装放到墙上"，本局把它推向极致：连 `cp_1` 都可以不要，只要拆家更快。

## 失误与改进

### 失误1: 侦察兵B `9acb25db` 整局未激活

**问题:** 初始侦察 `(-5,5)` 从第1轮到第11轮从未移动，不占 `cp_1`、不封锁HQ邻格、不参与输出——与 tg_0038 胜方复盘承认的失误完全同款。
**改进:** 第2-3轮应令其推进 `(-5,5)→(-3,2)→(-3,0)` 占领 `cp_1`：收入从6升至14，第二台重装/游骑兵的窗口大幅提前；即使不用于经济，也应在第5轮后移至 `(-3,1)` 充当HQ门卫。
**预期收益:** 据点分 0→30；若第4轮前占领，收入累计多约30-40补给，可支撑第8轮前后部署游骑兵形成双输出，击杀可能提前1-2轮。

### 失误2: 全程零据点，裁决结构偏科

**问题:** 终局 `controlPoints=0`，据点分0。本局靠第11轮速胜掩盖了偏科；若对手防守更好、对局拖入第30轮裁决，0据点对阵 `cp_2` 持有者的30分是结构性劣势。
**改进:** 危险距离的两个 supply 点距各自HQ仅1格，占点成本极低（侦察2次移动）。速攻流也应顺手拿 `cp_1`——它不是经济问题，是**裁决保险**。
**预期收益:** 裁决分+30，且 `cp_1` 提供部署点（邻接HQ），增加战术弹性。

### 失误3: 第7-10轮HQ邻格完全敞开，存在被偷家的暴露窗口

**问题:** 重装第7轮离开 `(-2,-1)` 深入口袋后，我方HQ三个开放邻格 `(-3,0)/(-2,-1)/(-3,1)` 长达4轮无人看守。敌侦察 `(1,2)` 距我HQ约8格（南线），若其放弃护航转而偷家，第10-11轮即可开始削我满血HQ（12伤/轮）。
**改进:** 本局对手选择了自己的走廊，暴露窗口未被惩罚；但规范做法是闪击出发时就规划"门卫交接"——侦察B提前进驻 `(-3,1)` 或 `(-3,2)`，重装离家后HQ邻格至少封锁1-2格。
**预期收益:** 消除唯一的翻车路径；若被偷家10-20伤，`ownHqHp` 分数受损且竞速压力陡增。

1. **tg_0038 的教训反向应验：** 该局WB（glm5.2）正是被Grok4.5以"中缝爆破+贴脸连刀"击败（460分惨败）；本局WB用同一条链 `(-1,-1)→…→(1,0)` 以彼之道还施彼身，且把承伤从23压到0。
2. **后手三连胜（tg_0036/0038/0040）证伪"先手必胜"：** 危险距离后手并非结构性必败，但三局胜法都是**让对手的重装变蠢**——前两局靠对手零爆破/慢爆破，本局更进一步，主动出击用诱饵**制造**对手重装的愚蠢。
3. **对 tg_0037 稳健路线的修正：** 该局用重装+游骑兵双输出14轮获胜；本局证明在对手中计的情况下纯重装3刀更快（11轮），但这是对手给的机会，不应视为通用最优——双输出仍是抗意外基准。
4. **新增一条本图规律：** `supplies=0` 权重下，**侦察兵的使命不是占点而是改变对手重装的用途**。tg_0038中侦察侧翼空转是败笔，本局侦察直插HQ是胜负手——差别在于是否让对手的行动点为你所用。

## 总结

### 胜利关键因素

1. **第1-3轮南线闪击**，后手主动打破对称，第3轮首血迫使对手全程带着HQ压力决策。
2. **第4-6轮诱饵战术**，一只38补给的侦察兵换走敌方重装2个席位回合+脱离爆破位，竞速时钟被偷走。

4. **读懂 `supplies=0` 与 `enemyHqDamage=20`**：无视经济差距，全部行动点押注HQ伤害。

### 核心战术原则

> **“对称攻城图，后手莫镜像；闪击逼回头，重装走最短。”**
