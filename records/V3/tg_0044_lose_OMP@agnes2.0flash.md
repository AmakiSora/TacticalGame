## 核心失败教训

### 1. 致命错误：没有尽早部署heavy破墙

**问题:** 我在第3轮和第6轮只部署了scout（各38），直到第9轮才deploy heavy（50）。而对手在第5轮就部署了heavy并爆破障碍。这6轮的差距让对手heavy从容推进到(1,0)直接攻击我的HQ。

```text
第1-4轮 / player_b: 只移动scout，零部署 —— 白白浪费了建立经济优势的时间窗口
第5轮 / player_a: deploy heavy(-3,1)，第6轮开始demolish(-2,1)
第9轮 / player_b: deploy heavy(2,1) —— 晚了4轮！此时对手heavy已在(1,0)威胁HQ
```

**改进:** 第1-2轮就应该攒钱部署heavy（约需4-5轮积累50补给=初始20+5×14=90）。实际上从第1回合就开始攒，第5-6轮应该已经堆够50补给deploy heavy，而不是第9轮。更优策略：**牺牲early scout数量，优先堆积heavy**。

**预期收益:** 如果早2轮部署heavy（第3-4轮），可以在对手heavy到达(1,0)之前就位于(1,0)附近拦截或同归于尽。

### 2. 战术失误：heavy部署后选择后撤而非拦截

**问题:** 第10轮和第11轮，我的heavy从(2,1)→(3,1)→(3,2)，不断远离敌方heavy(1,0)。敌方heavy距我HQ仅1格，每次攻击约33-35伤害。我应该把heavy移动到(2,0)挡路（但HQ挡路不可行）或爆破障碍靠近，而不是后退。

```text
第10轮：heavy(2,1)→(3,1)  [远离敌方heavy，距离增大]
第11轮：heavy(3,1)→(3,2)  [继续远离]
第12轮：enemy heavy(1,0)→attack HQ → HQ被摧毁
```

**改进:** 即使不能到(1,0)（可能被blocker阻挡），也应尝试爆破(2,0)附近的障碍打开通路。或者至少移动到能攻击敌方heavy的位置——heavy attackRange=1。

**预期收益:** 如果heavy能在(2,0)或(1,1)位置攻击敌方heavy，可以将其消灭或重创，保住HQ。

### 3. 战略失误：情报收集不足——未监控对手动向

**问题:** 从第5轮到第9轮，对手heavy已经从(-3,1)推到(1,-1)，我却完全没有采取任何防御措施。第7轮我还在用scout做无意义的前后移动(2,1)→(2,2)，对手却在悄悄逼近我的命门。

**改进:** 每次轮到我时，应先检查敌方单位最新位置和HQ血量，评估威胁等级再做决策。当对方heavy接近HQ半径2格内时应优先防御。

### 4. 决策失误：部署过多scout而忽视核心兵种

**问题:** 我总共部署了3个scout（成本3×38=114），但在这张地图上scout的攻击力(16)无法威胁到对手HQ（距离太远且attackRange=1）。而对手只有1个heavy就足以碾压全场。

```text
我的投入：3×38(scout)=114，零输出
对手投入：1×50(hheavy)=50，120HQ伤害
ROI对比：对手是绝对胜利
```

**改进:** 应部署1个heavy破墙 + 2个scout占点抢收入。放弃多余的scout部署。

## 总结

### 失败关键因素

1. **兵种选择错误**：部署3个scout(114补给)完全浪费了资源，这些单位在本图中没有任何战略价值（无法攻击任何目标，无法爆破障碍）
2. **heavy部署过晚**：第9轮才deploy heavy，比对手晚了至少4轮，导致对手heavy已经拆了我半条命
3. **防御决策失误**：heavy到位后选择后撤(3,1)→(3,2)而非拦截对手heavy，相当于眼睁睁看着HQ被拆
4. **先手优势浪费**：先手但没有用来抢占先机部署核心战力，而是浪费在scout的移动上

### 核心战术原则

> **"危险距离地图，heavy是唯一的刀——攒钱deploy heavy，爆破开路，直取HQ。别浪费补给的scout"**

### 一句话总结

**在对决型地图"危险距离"上，我用114补浪费在3个毫无用处的scout上，比对手晚4轮部署heavy，heavy到位后又选择后撤而非拦截，最终眼睁睁看着HQ在4次heavy攻击下被摧毁——这是一局教科书级的"选错核心兵种导致全面崩盘"。**
